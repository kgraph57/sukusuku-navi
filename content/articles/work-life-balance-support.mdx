# 愛育病院 小児科**おかもん先生** だより Vol.298

## 「仕事も育児も頑張りたい」――子育てと仕事の両立支援

こんにちは。愛育病院小児科の**おかもん先生**です。

「仕事を続けたいけど、子どもが頻繁に体調を崩します」「育休からの復帰が不安です」――子育てと仕事の両立は多くの保護者が抱える課題です。港区には**働く保護者を支援するさまざまな制度とサービス**があります [1]。

---

### Q1.「港区の保育サービスにはどのようなものがありますか？」

**お母さん**「保育園以外にも預け先はありますか？」

**おかもん先生**「港区には多様な保育サービスがあります [1]。」

| サービス | 対象 | 特徴 |
|---|---|---|
| **認可保育園** | 0歳〜就学前 | 区が認可、保育料は所得に応じる [1] |
| **認証保育所** | 0歳〜就学前 | 都が認証、延長保育に対応しやすい [1] |
| **小規模保育事業** | 0〜2歳 | 少人数で家庭的な保育 [1] |
| **家庭的保育（保育ママ）** | 0〜2歳 | 保育者の自宅で少人数保育 [1] |
| **事業所内保育** | 従業員の子ども | 職場の近くで保育 |
| **ベビーシッター利用支援** | 0歳〜就学前 | 費用の一部を助成 [1] |

| 港区の保育料の特徴 |
|---|
| 第2子以降の保育料が軽減 |
| 所得に応じた保育料設定 |
| 認証保育所の保育料補助あり |
| 3〜5歳児の保育料無償化 |

> **ポイント**
> - 港区は**多様な保育サービス**を提供 [1]
> - **認証保育所の補助**や**ベビーシッター支援**もある
> - **3〜5歳児の保育料は無償化**

---

### Q2.「子どもの急な病気に対応するサービスは？」

**お母さん**「子どもが急に熱を出した時に使えるサービスはありますか？」

**おかもん先生**「共働き家庭を支える複数のサービスがあります [1]。」

| サービス | 内容 | 利用方法 |
|---|---|---|
| **病児保育** | 病気の子どもを保育施設で預かる | 事前登録＋当日予約 [1] |
| **病後児保育** | 回復期の子どもを預かる | 事前登録＋当日予約 [1] |
| **ファミリー・サポート** | 地域の援助者が子どもを預かる | 事前登録 [1] |
| **ベビーシッター利用支援** | ベビーシッター費用を助成 | 対象事業者を利用 [1] |
| **子育て短期支援（ショートステイ）** | 保護者の病気等で一時的に預かる | 事前申請 [1] |

| 病児対応の準備チェックリスト |
|---|
| 病児保育施設に**事前登録**する |
| 複数の施設に登録しておく |
| ファミリー・サポートにも登録する |
| ベビーシッター事業者を把握する |
| 職場の**看護休暇制度**を確認する |
| **祖父母等のバックアップ体制**を整える |

> **ポイント**
> - **事前登録は元気な時に**済ませておく [1]
> - **複数のバックアップ体制**を準備する
> - 職場の**看護休暇制度**も活用する

---

### Q3.「働く親が使える法的な制度は？」

**お母さん**「法律で守られている制度はありますか？」

**おかもん先生**「**育児・介護休業法**で以下の制度が保障されています [2]。」

| 制度 | 内容 | 対象 |
|---|---|---|
| **育児休業** | 原則1歳まで（最長2歳まで延長可） [2] | 男女ともに |
| **育児短時間勤務** | 1日6時間の短時間勤務 [2] | 3歳未満の子がいる親 |
| **子の看護休暇** | 年5日（子が2人以上は10日） [2] | 小学校就学前の子がいる親 |
| **時間外労働の制限** | 残業の制限を請求できる [2] | 小学校就学前の子がいる親 |
| **深夜業の制限** | 深夜勤務の免除を請求できる [2] | 小学校就学前の子がいる親 |
| **産後パパ育休** | 出生後8週間以内に4週間まで [2] | 父親 |

| 知っておきたいポイント |
|---|
| 看護休暇は**時間単位で取得可能** [2] |
| 育児短時間勤務は**子が3歳になるまで** [2] |
| これらの制度の利用を理由に**不利益な扱いは禁止** [2] |
| パートタイム・契約社員でも利用可能（条件あり） |

> **ポイント**
> - **子の看護休暇**は小学校就学前まで年5日 [2]
> - **時間単位**で取得できるため、受診に便利
> - 制度利用による**不利益な扱いは法律で禁止** [2]

### Q4.「育休からの復帰に向けての準備は？」

**お母さん**「育休からの職場復帰が不安です」

**おかもん先生**「計画的な準備で復帰をスムーズにしましょう [3]。」

| 復帰の準備 | 時期の目安 |
|---|---|
| **保育園の申し込み** | 入園の6か月前〜 |
| **慣らし保育の確認** | 復帰の1〜2か月前 |
| **生活リズムの調整** | 復帰の1か月前〜 |
| **職場との面談** | 復帰の1〜2か月前 |
| **バックアップ体制の整備** | 復帰前に完了 |

| 復帰後の心構え |
|---|
| **最初の半年は病気ラッシュ**を覚悟する [3] |
| 完璧を求めすぎない |
| パートナーとの家事・育児分担を事前に話し合う |
| 周囲のサポートを積極的に使う |
| 自分の体調管理も大切にする |

> **ポイント**
> - **保育園入園後は風邪ラッシュ**を覚悟する [3]
> - **バックアップ体制**は復帰前に整える
> - **完璧を求めすぎない**ことが長期的な両立の鍵

### Q5.「港区独自の両立支援はありますか？」

**お母さん**「港区ならではの支援制度はありますか？」

**おかもん先生**「港区は働く保護者への支援が充実しています [1]。」

| 港区の支援 | 内容 |
|---|---|
| **延長保育** | 最大22時まで対応の園もある [1] |
| **休日保育** | 日曜・祝日の保育 [1] |
| **夜間保育** | 夜間の就労に対応 [1] |
| **学童クラブ** | 小学生の放課後の居場所 [1] |
| **放課GO→** | 港区独自の放課後支援 [1] |
| **ベビーシッター利用支援** | 費用の一部を助成 [1] |

| 港区の子育て情報を得る方法 |
|---|
| **港区公式ウェブサイト** |
| **港区子育て応援アプリ** |
| **各総合支所の子ども家庭支援センター** |
| **保育コンシェルジュ**（保育の相談窓口） |

> **ポイント**
> - 港区は**延長保育、休日保育、夜間保育**も充実 [1]
> - **保育コンシェルジュ**に保育の相談ができる
> - **子育て応援アプリ**で最新情報をチェック
