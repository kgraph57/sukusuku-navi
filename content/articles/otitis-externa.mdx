# 愛育病院 小児科**おかもん先生** だより Vol.206

## 「耳がかゆい、痛い」――外耳炎

こんにちは。愛育病院小児科の**おかもん先生**です。

外耳炎は、外耳道（耳の穴から鼓膜までの部分）に炎症が起こる病気です。プールの後や耳かきのしすぎで起こることが多く、「スイマーズイヤー」とも呼ばれます。今回は、小児の外耳炎についてお伝えします。

---

### Q1.「外耳炎とは？」

**お母さん**「プールの後から耳が痛いと言っています」

**おかもん先生**「外耳炎は**外耳道の皮膚に細菌や真菌が感染して炎症を起こす**病気です [1]。」

| 基本情報 | 詳細 |
|---|---|
| **原因** | 細菌感染（緑膿菌、黄色ブドウ球菌が多い） |
| **誘因** | 水泳、耳かき、湿った環境 |
| **症状** | 耳痛、かゆみ、耳漏 |
| **特徴** | 耳たぶを引っ張ると痛い |

| 中耳炎との違い |
|---|
| 外耳炎：耳を引っ張ると痛い。鼓膜の外側 |
| 中耳炎：耳を引っ張っても痛くない。鼓膜の内側 |

> **ポイント**
> - 耳たぶを引っ張ると痛いのが特徴
> - プールや耳かきが誘因
> - 中耳炎とは別の病気

---

### Q2.「原因は何ですか？」

**お母さん**「なぜ外耳炎になるのですか？」

**おかもん先生**「外耳道の皮膚バリアが壊れることが原因です [2]。」

| 原因 | メカニズム |
|---|---|
| **耳かきのしすぎ** | 外耳道の皮膚を傷つける |
| **水泳・プール** | 水分で皮膚がふやける |
| **イヤホンの長時間使用** | 外耳道の湿度が上がる |
| **アトピー性皮膚炎** | 皮膚バリア機能の低下 |
| **異物** | 小さなおもちゃ等が入る |

「**耳かきは基本的に不要**です。耳垢は自然に排出されます。耳かきによる外耳道の損傷が外耳炎の最大の原因です。」

> **ポイント**
> - 耳かきのしすぎが最大の原因
> - 耳垢は自然に排出される
> - 水泳後は耳を乾かす

---

### Q3.「治療はどうしますか？」

**お母さん**「どう治療しますか？」

**おかもん先生**「点耳薬（耳に入れる薬）が治療の基本です [3]。」

| 治療 | 内容 |
|---|---|
| **点耳薬** | 抗菌薬＋ステロイドの点耳薬が標準 |
| **耳の清掃** | 医師が外耳道の清掃を行う |
| **鎮痛剤** | 痛みにはアセトアミノフェン |
| **重症の場合** | 内服抗菌薬の追加 |

| 点耳薬の使い方 |
|---|
| 患側の耳を上にして横になる |
| 処方された滴数を耳に垂らす |
| 5分間そのままの姿勢を保つ |
| 1日2-3回、7-10日間続ける |

> **ポイント**
> - 点耳薬が治療の基本
> - 正しい使い方が大切
> - 7-10日間は続ける

---

### Q4.「プールに入っても大丈夫ですか？」

**お母さん**「治るまでプールは休むべきですか？」

**おかもん先生**「**治療中はプールは控えてください** [4]。」

| 対応 | 詳細 |
|---|---|
| **治療中** | プール・水泳は禁止 |
| **治癒後** | 医師の許可を得てから再開 |
| **予防** | 水泳後は耳を乾かす |
| **耳栓** | 必要に応じて使用 |

「水泳を再開する際は、**水泳後にタオルで耳を拭き、頭を傾けて水を出す**習慣をつけてください。」

> **ポイント**
> - 治療中はプールは休む
> - 水泳後は耳を乾かす習慣を
> - 耳栓の使用も検討

---

### Q5.「耳かきはどうすればいいですか？」

**お母さん**「耳掃除は全くしなくていいのですか？」

**おかもん先生**「**基本的に耳かきは不要**です [5]。」

| 耳垢について | 詳細 |
|---|---|
| **自然排出** | 耳垢は自然に外に出てくる |
| **保護作用** | 耳垢は外耳道を保護する役割がある |
| **耳かきの害** | 奥に押し込む、外耳道を傷つける |
| **綿棒も注意** | 外耳道の入り口だけ軽く拭く程度に |
| **耳垢栓塞** | 詰まった場合は耳鼻科で除去 |

「**耳垢が気になる場合は耳鼻科で除去**してもらってください。家庭での耳かきは外耳炎の原因になります。」

> **ポイント**
> - 耳かきは基本的に不要
> - 耳垢には保護作用がある
> - 気になる場合は耳鼻科へ

---

## 今号のまとめ

- **外耳炎は外耳道の炎症**。耳を引っ張ると痛い
- **耳かきのしすぎが最大の原因**
- **点耳薬が治療の基本**
- **治療中はプールは休む**
- **耳かきは基本的に不要**。耳垢は自然に出る
