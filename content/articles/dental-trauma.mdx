# 愛育病院 小児科おかもん だより Vol.317

## 「転んで前歯をぶつけた！」――乳歯の外傷、親が知るべき対応

> **今号のポイント**
>
> 1. 乳歯の外傷は1〜3歳に多い。出血時はまずガーゼで圧迫止血を
> 2. 乳歯が抜けても再植は不要。永久歯が抜けた場合は牛乳に入れて30分以内に歯科へ
> 3. ぶつけた後の歯の変色は歯髄壊死のサイン。永久歯への影響もあるため歯科でフォローを

こんにちは。愛育病院小児科のおかもん先生です。

**「転んで前歯をぶつけて血が出ています」「歯がグラグラしているのですが大丈夫ですか？」「ぶつけた歯が黒くなってきました」**――歩き始めたお子さんを持つ保護者の方から、非常に多くいただくご相談です。

1〜3歳の子どもは頭が重く、バランス感覚も未熟なため、転倒して前歯（上の前歯が最多）をぶつけることがとても多いです [1][2]。慌ててしまうお気持ちはよくわかりますが、**適切な初期対応と受診のタイミング**を知っておけば安心です。

---

### Q1.「歯をぶつけて出血しています。まず何をすればいいですか？」

**お母さん**「子どもが転んで口から血が出ています。歯が折れたかもしれません。どうすればいいですか？」

**おかもん先生**「まず**落ち着いてください**。口の中の出血は唾液と混じるため、実際より多く見えることがほとんどです [1]。以下の順序で対応してください」

| ステップ | 対応 |
|---|---|
| **1. 止血** | 清潔なガーゼやハンカチで出血部位を**5〜10分間しっかり圧迫** [1][3] |
| **2. 口の中を確認** | 出血が落ち着いたら、歯が折れていないか、グラついていないか、歯茎の傷を確認 [1] |
| **3. 歯の破片を探す** | 歯が欠けている場合、破片があれば拾って保管（接着できる場合がある）[3] |
| **4. 冷やす** | 唇や歯茎が腫れている場合、外側から冷たいタオルで冷やす [1] |
| **5. 受診判断** | 下記の受診基準を参考に、歯科（または小児歯科）を受診 [1][2] |

**お母さん**「出血がなかなか止まりません」

**おかもん先生**「口の中は血流が豊富なので出血が目立ちますが、**圧迫を10分以上続ければほとんどの場合止まります** [1]。ガーゼを当てて噛ませるか、指で押さえてください。それでも止まらない場合は、歯科または救急を受診してください」

| 受診の緊急度 | 状況 |
|---|---|
| **すぐに受診（当日）** | 歯が抜けた、大きく折れた、ぐらつきが強い、歯の位置がずれた、出血が止まらない [1][2] |
| **翌日までに受診** | 歯が少しグラつく、小さく欠けた、歯茎の腫れ [2] |
| **1週間以内に受診** | ぶつけたが見た目に異常なし（念のため確認）[2] |

> **ポイント**
> - 口の出血は**ガーゼで5〜10分間圧迫止血**が基本 [1][3]
> - 出血は唾液と混じり多く見えるが、多くは圧迫で止まる [1]
> - 歯が抜けた・大きく折れた・位置がずれた場合は**当日中に歯科受診** [1][2]

### Q2.「歯がグラグラしています。抜けそうで心配です」

**お母さん**「ぶつけた前歯がグラグラしています。抜けてしまいませんか？」

**おかもん先生**「歯をぶつけた後のグラつきにはいくつかの段階があります [2][4]。程度によって対応が異なります」

| 外傷の分類 | 状態 | 対応 |
|---|---|---|
| **振盪（しんとう）** | 歯に衝撃を受けたが、グラつきなし。触ると痛い [4] | 経過観察でOK [4] |
| **亜脱臼（あだっきゅう）** | 歯が**少しグラグラ**するが、位置は正常 [4] | 軟らかい食事で経過観察。1〜2週間で安定することが多い [2][4] |
| **側方脱臼** | 歯が**前後や横にずれている** [4] | **歯科受診が必要**。整復・固定を行う [4] |
| **挺出（ていしゅつ）** | 歯が**上に飛び出している**（長くなったように見える）[4] | **歯科受診が必要** [4] |
| **陥入（かんにゅう）** | 歯が**歯茎の中に押し込まれている** [4] | **歯科受診が必要**。永久歯胚への影響を確認 [2][4] |
| **完全脱臼（脱落）** | 歯が**完全に抜けた** [4] | 乳歯→再植は通常不要。永久歯→緊急対応（Q3参照）[2][5] |

**おかもん先生**「軽度のグラつき（亜脱臼）であれば、**無理に触らず、軟らかい食事にして1〜2週間様子を見る**のが基本です [2][4]。多くの場合、歯は自然に安定してきます。ただし、歯並びに影響が出そうな場合や、痛みが強い場合は歯科を受診してください」

**お母さん**「陥入（歯茎に埋まってしまった場合）は怖いですね」

**おかもん先生**「はい。**陥入は乳歯外傷の中でも最も注意が必要な外傷**です [2][4]。歯が歯茎の中に押し込まれることで、**その下にある永久歯の芽（歯胚）を傷つける可能性**があるからです [2][6]。陥入を認めた場合は必ず歯科を受診し、レントゲンで永久歯胚の位置と状態を確認してもらってください [4]」

> **ポイント**
> - 軽度のグラつきは**軟らかい食事で経過観察**。多くは1〜2週間で安定 [2][4]
> - 歯の位置がずれた（側方脱臼・挺出・陥入）場合は**歯科受診が必要** [4]
> - **陥入は永久歯胚への影響に注意**。必ずレントゲンで確認を [2][6]

### Q3.「歯が抜けてしまいました。どうすればいいですか？」

**お母さん**「転んだ拍子に前歯が完全に抜けてしまいました！」

**おかもん先生**「ここで**乳歯と永久歯で対応がまったく異なる**ことを知っておいてください [2][5]」

| 状況 | 対応 |
|---|---|
| **乳歯が抜けた場合** | **再植（元に戻す処置）は通常行いません** [2][5] |
| **永久歯が抜けた場合** | **緊急事態！30分以内に歯科を受診** [5][7] |

**お母さん**「乳歯は戻さなくていいんですか？」

**おかもん先生**「はい。**乳歯を再植しない理由**は以下の通りです [2][5]」

| 理由 | 説明 |
|---|---|
| **永久歯胚への損傷リスク** | 再植の処置自体が、その下にある発育中の永久歯を傷つける可能性がある [2][5] |
| **感染リスク** | 再植した乳歯が感染を起こし、永久歯胚に波及する可能性 [5] |
| **吸収リスク** | 再植した乳歯が癒着や異常吸収を起こし、永久歯の萌出を妨げる可能性 [5] |
| **永久歯が生えてくる** | いずれ永久歯に生え替わるため、乳歯の喪失は長期的には問題にならないことが多い [2] |

**おかもん先生**「一方、**永久歯が抜けた場合は緊急事態**です [5][7]。以下の対応を覚えておいてください」

| 永久歯が抜けた時の対応 | 具体的方法 |
|---|---|
| **歯を拾う** | **歯の根（下の部分）は触らない**。冠部（上の白い部分）を持つ [5][7] |
| **汚れを落とす** | 水道水で**軽くすすぐ程度**（10秒以内）。ゴシゴシ洗わない [5] |
| **保存液に入れる** | **牛乳**がベスト。なければ**口の中（頬と歯茎の間）**に含む [5][7] |
| **30分以内に歯科へ** | **時間が勝負**。再植の成功率は時間とともに低下する [5][7] |

| 保存液の優先順位 | 理由 |
|---|---|
| **1. 歯牙保存液（デントサプライなど）** | 歯根膜細胞の生存に最適 [7] |
| **2. 牛乳** | 浸透圧がちょうどよく、歯根膜細胞を6時間程度保護できる [5][7] |
| **3. 生理食塩水** | 牛乳がない場合の代替 [7] |
| **4. 口の中（唾液）** | 他に何もない場合。誤飲に注意 [5] |
| **NG: 水道水** | 低張液のため歯根膜細胞が死滅する [5][7] |
| **NG: ティッシュに包む** | 乾燥により歯根膜が壊死する [5] |

> **ポイント**
> - **乳歯が抜けた場合は再植不要**。永久歯胚を守ることが優先 [2][5]
> - **永久歯が抜けたら牛乳に入れて30分以内に歯科へ** [5][7]
> - 歯の根は触らない。水道水に長く浸さない [5][7]

### Q4.「ぶつけた歯が黒く変色してきました。永久歯に影響はありますか？」

**お母さん**「2週間前にぶつけた歯が、だんだん茶色〜黒っぽく変色してきました。大丈夫ですか？」

**おかもん先生**「歯の変色は、**歯髄（しずい＝歯の中の神経と血管）が損傷を受けたサイン**です [2][6][8]。外傷後の変色には2つのパターンがあります」

| 変色の色 | 意味 | 経過 |
|---|---|---|
| **ピンク〜赤みがかった色** | 歯髄内の**出血**。血液成分が象牙質に浸透 [6][8] | 数週間〜数ヶ月で回復することもある [8] |
| **灰色〜黒色** | **歯髄壊死（神経が死んだ状態）**の可能性が高い [6][8] | 自然回復は難しいことが多い [8] |
| **黄色** | 歯髄が反応して**象牙質を過剰に形成**している [8] | 歯の生活力は保たれていることが多い [8] |

**おかもん先生**「灰色〜黒色に変色した場合、多くは**歯髄壊死**を起こしています [6][8]。歯髄壊死した乳歯は、そのままでも問題ないことが多いですが、以下の場合は歯科での処置が必要です」

| 処置が必要なサイン | 理由 |
|---|---|
| **歯茎にニキビのような腫れ（フィステル）** | 歯根の先に膿が溜まっている（根尖性歯周炎）[6][8] |
| **歯茎の腫れ・痛み** | 感染の広がり [8] |
| **歯の動揺が進行** | 歯根の吸収が進んでいる可能性 [6] |

**お母さん**「永久歯への影響が心配です」

**おかもん先生**「乳歯の外傷が**永久歯に影響を与える可能性**はあります [2][6][9]。特に注意が必要なのは以下のケースです」

| 永久歯への影響 | 説明 | 頻度 |
|---|---|---|
| **エナメル質の白斑・形成不全** | 永久歯の表面に白い斑点やくぼみができる [6][9] | 比較的多い |
| **永久歯の萌出方向の異常** | 永久歯が曲がって生えてくる [9] | まれ |
| **永久歯の萌出遅延** | 生え替わりが遅れる [6] | まれ |
| **永久歯胚の壊死** | 永久歯そのものが発育しない（非常にまれ）[9] | 非常にまれ |

**おかもん先生**「影響の程度は、**外傷を受けた時の年齢と外傷の種類**によります [6][9]。年齢が低いほど（特に3歳以下）、永久歯胚がまだ発育の初期段階にあるため、影響を受けやすいとされています [9]。また、**陥入（歯が歯茎に押し込まれた場合）が最も永久歯への影響リスクが高い**です [2][6]」

**おかもん先生**「このため、乳歯をぶつけた場合は、たとえ当座の問題がなさそうでも、**定期的に歯科でフォロー**してもらうことをお勧めします [2][8]。レントゲンで永久歯胚の状態を確認し、問題があれば早期に対応できます」

> **ポイント**
> - 外傷後の**灰色〜黒色の変色**は歯髄壊死のサイン [6][8]
> - 歯茎の腫れ（フィステル）が出たら歯科受診を [6][8]
> - **3歳以下の陥入**は永久歯への影響リスクが最も高い [2][6][9]
> - 乳歯外傷後は**定期的な歯科フォロー**が重要 [2][8]

## まとめ

- 乳歯の外傷は**1〜3歳に多い**。転倒による上の前歯の損傷が最多 [1][2]
- 出血時は**ガーゼで5〜10分間圧迫止血** [1][3]
- グラつきが軽度なら**軟らかい食事で経過観察**。位置がずれたら歯科へ [2][4]
- **乳歯が抜けても再植は不要**。永久歯胚を守ることが優先 [2][5]
- **永久歯が抜けたら牛乳に入れて30分以内に歯科へ**（緊急事態）[5][7]
- 歯の**灰色〜黒色の変色は歯髄壊死**のサイン。歯科でフォローを [6][8]
- **3歳以下の陥入は永久歯への影響リスクが高い**。レントゲンで確認を [2][6][9]
