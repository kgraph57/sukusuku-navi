# 愛育病院 小児科**おかもん先生** だより Vol.256

## 「鼻に何か入れました！」――鼻・耳の異物

こんにちは。愛育病院小児科の**おかもん先生**です。

幼児は好奇心から鼻や耳に小さなものを入れてしまうことがあります。ビーズ、消しゴム、小さなおもちゃの部品などが多く、適切に対応しないと感染や損傷の原因になります。今回は、鼻と耳の異物についてお伝えします。

---

### Q1.「鼻に異物が入った場合は？」

**お母さん**「子どもが鼻にビーズを入れてしまいました」

**おかもん先生**「鼻の異物は**早めの受診**が大切です [1]。」

| 基本情報 | 詳細 |
|---|---|
| **好発年齢** | 2-5歳 |
| **多い異物** | ビーズ、消しゴム、紙、食べ物、ボタン電池 |
| **症状** | 片側の鼻水、臭い鼻汁、鼻出血 |
| **特に危険** | ボタン電池（化学熱傷を起こす） |

| 家庭での対応 |
|---|
| 反対の鼻を押さえて「フン」と強く鼻をかませる |
| 無理に取ろうとしない（奥に押し込む危険） |
| ピンセットなどは使わない |
| 取れなければ耳鼻科を受診 |

「**ボタン電池が鼻に入った場合は緊急**です。数時間で鼻の粘膜が壊死するため、すぐに受診してください。」

> **ポイント**
> - 片側だけの臭い鼻水は異物を疑う
> - 無理に取ろうとしない
> - ボタン電池は緊急

---

### Q2.「耳に異物が入った場合は？」

**お母さん**「耳にビーズが入ったようです」

**おかもん先生**「耳の異物も**無理に取ろうとせず受診**してください [2]。」

| 耳の異物 | 詳細 |
|---|---|
| **多い異物** | ビーズ、消しゴム、虫、綿棒の先 |
| **症状** | 耳の痛み、聞こえにくさ、耳だれ |
| **虫の場合** | ゴソゴソと動く感覚、痛み |

| 対応 |
|---|
| 無理に取ろうとしない（外耳道を傷つける） |
| 綿棒やピンセットで奥に押し込まない |
| 虫の場合：暗い部屋で懐中電灯を当てる（虫が光に向かう） |
| 虫が出なければオリーブオイルを数滴入れて殺虫してから受診 |

> **ポイント**
> - 無理に取ろうとしない
> - 虫は光でおびき出すか油で殺虫
> - 耳鼻科を受診

---

### Q3.「気づかないこともある？」

**お母さん**「異物を入れたことに気づかないこともありますか？」

**おかもん先生**「はい、**入れたことを言わない、忘れている**ことがあります [3]。」

| 気づくきっかけ | 詳細 |
|---|---|
| **片側の鼻水** | 片方だけ臭い鼻水が続く |
| **片側の鼻出血** | 繰り返す片側の鼻血 |
| **耳だれ** | 片側の耳から膿が出る |
| **口臭** | 鼻の異物による悪臭 |

「**片側だけの症状**（鼻水、鼻血、耳だれ）が続く場合は、異物を疑ってください。」

> **ポイント**
> - 子どもは入れたことを言わないことがある
> - 片側だけの症状は異物のサイン
> - 臭い鼻水が続いたら受診

---

### Q4.「気道異物（のどに詰まった場合）は？」

**お母さん**「何か飲み込んでのどに詰まったら？」

**おかもん先生**「気道異物は**窒息につながる緊急事態**です [4]。」

| 症状 | 詳細 |
|---|---|
| **突然の咳** | 食事中やおもちゃで遊んでいる時に突然咳込む |
| **喘鳴** | ゼーゼー、ヒューヒュー |
| **チアノーゼ** | 顔色が紫色に |
| **声が出ない** | 完全閉塞のサイン |

| 窒息時の応急処置 |
|---|
| **1歳未満**：背部叩打法（うつ伏せにして背中を5回叩く）＋胸部突き上げ法 |
| **1歳以上**：腹部突き上げ法（ハイムリック法） |
| 咳ができるなら咳を続けさせる |
| 意識がなくなったら心肺蘇生＋119番 |

> **ポイント**
> - 食事中の突然の咳は気道異物を疑う
> - 1歳未満は背部叩打法
> - 1歳以上はハイムリック法

---

### Q5.「異物事故の予防は？」

**お母さん**「異物事故を防ぐにはどうすればいいですか？」

**おかもん先生**「**小さなものを手の届かない場所に保管**することが基本です [5]。」

| 予防策 | 内容 |
|---|---|
| **小さなもの** | トイレットペーパーの芯（直径39mm）を通るものは危険 |
| **食品** | ミニトマト、ぶどう、ナッツ類は4つ切りに |
| **おもちゃ** | 対象年齢を守る。上の子のおもちゃに注意 |
| **ボタン電池** | テープで絶縁、使用後すぐに廃棄 |

| 窒息リスクの高い食品 |
|---|
| ミニトマト（4つ切りに） |
| ぶどう（4つ切りに） |
| ナッツ類（3歳まで避ける） |
| 餅・白玉団子 |
| りんご（薄切りに） |

> **ポイント**
> - 直径39mm以下のものは手の届かない場所に
> - 丸い食品は4つ切りに
> - ナッツ類は3歳まで避ける

## 今号のまとめ

- **鼻・耳の異物は無理に取ろうとしない**
- **ボタン電池は鼻でも耳でも緊急**
- **片側だけの症状**は異物を疑う
- **気道異物は窒息の緊急事態**
- **小さなものを手の届かない場所に保管**
