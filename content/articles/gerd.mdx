# 愛育病院 小児科**おかもん先生** だより Vol.190

## 「いつも吐き戻します」――胃食道逆流症（GERD）

こんにちは。愛育病院小児科の**おかもん先生**です。

赤ちゃんの吐き戻しは非常に一般的で、生後数か月の赤ちゃんの約50%に見られます。多くは生理的なもので自然に改善しますが、中には治療が必要な胃食道逆流症（GERD）のケースもあります。今回は、生理的な逆流とGERDの違いについてお伝えします。

---

### Q1.「吐き戻しは病気ですか？」

**お母さん**「毎回ミルクを吐き戻します。大丈夫ですか？」

**おかもん先生**「多くの場合、乳児の吐き戻しは**生理的な胃食道逆流（GER）**で、病気ではありません [1]。」

| 項目 | 生理的逆流（GER） | 胃食道逆流症（GERD） |
|---|---|---|
| **頻度** | 乳児の約50% | 乳児の約5% |
| **体重増加** | 正常 | 不良 |
| **機嫌** | 良好 | 不機嫌、泣きが多い |
| **合併症** | なし | 食道炎、誤嚥性肺炎等 |
| **自然経過** | 1歳までに95%が改善 | 治療が必要 |

> **ポイント**
> - 乳児の吐き戻しの多くは生理的
> - 体重が順調に増えていれば心配ない
> - 1歳までに95%が自然に改善

---

### Q2.「GERDを疑うサインは？」

**お母さん**「どんな時に病院に行くべきですか？」

**おかもん先生**「以下のサインがあればGERDの可能性があります [2]。」

| 受診すべきサイン | 説明 |
|---|---|
| **体重が増えない** | 嘔吐による栄養不足 |
| **授乳を嫌がる** | 食道の痛みで飲みたがらない |
| **反り返り** | 背中を反らせて泣く（食道の痛み） |
| **血液混じりの嘔吐** | 食道炎の可能性 |
| **繰り返す肺炎** | 誤嚥の可能性 |
| **無呼吸エピソード** | 逆流による気道反射 |
| **緑色の嘔吐（胆汁性）** | 腸閉塞等の除外が必要（緊急） |

「**胆汁性嘔吐（緑色の嘔吐）**は緊急事態です。GERDではなく腸閉塞の可能性があるため、すぐに受診してください。」

> **ポイント**
> - 体重増加不良が最も重要なサイン
> - 反り返って泣くのは食道の痛みの可能性
> - 緑色の嘔吐は緊急受診

---

### Q3.「GERDの治療は？」

**お母さん**「治療が必要な場合、どうしますか？」

**おかもん先生**「段階的に治療を進めます [3]。」

| 治療段階 | 内容 |
|---|---|
| **①生活指導** | 授乳後の体位（上体挙上）、少量頻回授乳 |
| **②ミルクの工夫** | ARミルク（とろみつき）への変更 |
| **③薬物療法** | 制酸薬（PPI）、アルギン酸塩 |
| **④手術** | 重症で薬物療法に反応しない場合（まれ） |

| 家庭でできる対策 |
|---|
| 授乳後30分は上体を起こした姿勢を保つ |
| ゲップをしっかりさせる |
| 少量ずつ頻回に授乳する |
| おむつをきつく締めすぎない |
| 寝かせる時は頭側を少し高くする |

> **ポイント**
> - まず生活指導から開始
> - ARミルクが有効な場合がある
> - 薬物療法は医師の判断で

---

### Q4.「年長児のGERDもありますか？」

**お母さん**「小学生の子どもが胸焼けを訴えます」

**おかもん先生**「年長児・学童にもGERDは見られます [4]。」

| 年齢 | 主な症状 |
|---|---|
| **乳児** | 嘔吐、体重増加不良、反り返り |
| **幼児** | 嘔吐、食事拒否、腹痛 |
| **学童** | 胸焼け、胸の痛み、嚥下時の痛み |
| **思春期** | 成人と同様の症状（胸焼け、酸っぱいものが上がってくる） |

「年長児の場合、**肥満、炭酸飲料、遅い夕食、食後すぐに横になる**などが悪化因子になります。」

> **ポイント**
> - 年長児にもGERDは見られる
> - 年齢により症状が異なる
> - 生活習慣の改善が重要

---

### Q5.「GERDの長期的な見通しは？」

**お母さん**「ずっと薬を飲み続けなければいけませんか？」

**おかもん先生**「乳児のGERDの多くは**1歳までに自然に改善**します [5]。」

| 年齢 | 経過 |
|---|---|
| **1歳まで** | 約95%の乳児の逆流が改善 |
| **離乳食開始後** | 固形食の摂取で改善することが多い |
| **学童期のGERD** | 生活習慣の改善＋薬物療法で多くが管理可能 |

「薬物療法は**症状が改善したら漸減・中止**を試みます。長期的にPPIを継続する必要があるケースはまれです。」

> **ポイント**
> - 乳児のGERDは1歳までにほとんどが改善
> - 離乳食開始が改善のきっかけになることも
> - 薬は症状改善後に漸減・中止を試みる

## 今号のまとめ

- **体重増加不良があればGERD**を疑い受診

- **緑色の嘔吐は緊急受診**
- **1歳までに95%が改善**します
