# 愛育病院 小児科**おかもん先生** だより Vol.282

## 「わざとじゃないんです」――チック症の正しい理解

こんにちは。愛育病院小児科の**おかもん先生**です。

「子どもが頻繁にまばたきをするんです」「首をカクカク動かすのが気になります」――チック症は**小児の約15〜20%が一過性に経験する**非常にありふれた症状です [1]。しかし、「わざとやっている」「やめなさい」と注意してしまう保護者の方が少なくありません。チックは**不随意運動であり、本人の意思でコントロールできるものではない**ことを知っていただきたいと思います。

---

### Q1.「チック症とはどんな病気ですか？」

**お母さん**「チックってどういうものですか？」

**おかもん先生**「チックとは**突発的、急速、反復的、非律動的な運動または発声**のことです [2]。本人の意思とは関係なく出現します。」

| 項目 | 内容 |
|---|---|
| **有病率** | 小児の約15〜20%が一過性に経験 [1] |
| **好発年齢** | 4〜6歳に発症が多い [1] |
| **男女比** | 男児に多い（約3〜4:1） [1] |
| **ピーク** | 10〜12歳で症状が最も強くなることが多い [3] |
| **自然経過** | 約70〜80%は思春期後半までに軽快 [3] |

| チックの種類 | 具体例 |
|---|---|
| **単純運動チック** | まばたき、首振り、肩すくめ、顔しかめ [2] |
| **複雑運動チック** | 飛び跳ね、物に触る、回転する [2] |
| **単純音声チック** | 咳払い、鼻すすり、うなり声 [2] |
| **複雑音声チック** | 単語の反復、状況に合わない言葉 [2] |

> **ポイント**
> - チックは**不随意運動**であり、わざとではない [2]
> - 小児の**約15〜20%**が一過性に経験する一般的な症状 [1]
> - 多くは**思春期後半までに自然に軽快**する [3]

---

### Q2.「チック症の分類を教えてください」

**お母さん**「チック症にも種類があるのですか？」

**おかもん先生**「DSM-5-TRでは持続期間と種類によって分類されます [2]。」

| 分類 | 基準 | 頻度 |
|---|---|---|
| **暫定的チック症** | 運動チックまたは音声チックが1年未満 [2] | 最も多い |
| **持続性運動または音声チック症** | 運動チックまたは音声チックが1年以上 [2] | やや少ない |
| **トゥレット症** | 運動チック＋音声チックの両方が1年以上 [2] | 約0.3〜1% [1] |

| チックの経過の特徴 |
|---|
| 症状は**増減を繰り返す**（waxing and waning） |
| **ストレス、疲労、興奮**で悪化しやすい [3] |
| **集中している時**は目立たないことが多い |
| **睡眠中**は消失または軽減する [3] |
| チックの種類は**時期によって変化**する |

> **ポイント**
> - 多くは**暫定的チック症**で1年以内に消失 [2]
> - **トゥレット症**は運動＋音声チックが1年以上持続 [2]
> - 症状は増減を繰り返すのが特徴 [3]

---

### Q3.「チックの原因は何ですか？」

**お母さん**「ストレスが原因ですか？ 私の育て方が悪いのでしょうか？」

**おかもん先生**「チック症は**脳の大脳基底核を中心とした神経回路の機能異常**が原因です。育て方やストレスが原因ではありません [4]。」

| 要因 | 内容 |
|---|---|
| **遺伝的要因** | 遺伝率は約50〜60% [4] |
| **神経生物学的要因** | 大脳基底核のドパミン系の過活動 [4] |
| **環境的要因** | ストレスは悪化因子であるが原因ではない [3] |
| **併存症の影響** | ADHD、OCDが併存しやすい [5] |

「ストレスは**チックを悪化させる要因**にはなりますが、**チックの原因ではありません** [3]。お母さんの育て方のせいでもありません。」

> **ポイント**
> - チック症は**脳の神経回路の機能異常**が原因 [4]
> - 育て方やストレスが原因ではない
> - ストレスは悪化因子にはなりうる [3]

---

### Q4.「治療は必要ですか？」

**お母さん**「チックは治療すべきですか？」

**おかもん先生**「**軽症のチックは治療不要で、経過観察が基本**です [3]。」

| 重症度 | 対応 |
|---|---|
| **軽症**（日常生活に支障なし） | 経過観察、心理教育 [3] |
| **中等症**（社会生活に一部支障） | 行動療法（CBIT/HRT） [6] |
| **重症**（著しい機能障害） | 行動療法＋薬物療法 [6] |

| 治療法 | 内容 |
|---|---|
| **心理教育** | チックの仕組みを家族・学校に説明 [3] |
| **CBIT（包括的行動療法）** | チックに対する行動介入で最もエビデンスあり [6] |
| **HRT（ハビットリバーサル）** | チックに気づき、拮抗する動作を行う訓練 [6] |
| **薬物療法** | 重症例にα2作動薬、抗精神病薬 [3] |

> **ポイント**
> - 軽症では**治療不要、経過観察**が基本 [3]
> - 治療が必要な場合は**行動療法（CBIT）が第一選択** [6]
> - 「やめなさい」と注意するのは逆効果

---

### Q5.「家庭や学校での対応は？」

**お母さん**「周囲にはどう説明すればよいですか？」

**おかもん先生**「**チックを指摘しない、注意しない**ことが最も重要です [3]。」

| 推奨される対応 | 具体的な方法 |
|---|---|
| **チックを指摘しない** | 気づいても知らないふりをする |
| **リラックスできる環境** | 過度なプレッシャーを避ける |
| **十分な睡眠** | 疲労はチックを悪化させる |
| **学校への説明** | 担任にチック症について情報提供 |
| **からかいへの対応** | いじめにつながらないよう学校と連携 |

| 避けるべき対応 |
|---|
| 「やめなさい」と注意する |
| チックのたびに反応する |
| 「気合で止められるでしょ」と言う |
| チックを真似する |

> **ポイント**
> - チックを**指摘・注意しない**ことが最も大切 [3]
> - 学校への情報提供で**からかい防止**を
> - **十分な睡眠と適度なストレス管理**が症状緩和に

## 今号のまとめ

- チックは**不随意運動**であり、わざとではない
- 多くは**思春期後半までに自然に軽快**する
- 軽症では**治療不要、経過観察**が基本
- **「やめなさい」と注意するのは逆効果**
