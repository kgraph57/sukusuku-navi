# 愛育病院 小児科**おかもん先生** だより Vol.283

## 「何度も手を洗わずにいられない」――強迫性障害（OCD）

こんにちは。愛育病院小児科の**おかもん先生**です。

「手を何十回も洗わないと気が済みません」「物の並べ方にこだわって、ずれると泣きます」――こうした行動が過度になり、日常生活に支障をきたしている場合、**強迫性障害（OCD: Obsessive-Compulsive Disorder）**の可能性があります。小児のOCDは**約1〜3%**に認められ、適切な治療で改善が期待できます [1]。

---

### Q1.「強迫性障害とはどんな病気ですか？」

**お母さん**「こだわりが強いのと強迫性障害は違うのですか？」

**おかもん先生**「強迫性障害は、**不合理だとわかっていても止められない考え（強迫観念）と行動（強迫行為）**が特徴です [2]。」

| 項目 | 内容 |
|---|---|
| **有病率** | 小児の約1〜3% [1] |
| **好発年齢** | 7〜12歳に多い（二峰性：小児期と青年期） [1] |
| **男女比** | 小児期は男児にやや多い、思春期以降はほぼ同等 [1] |
| **経過** | 治療なしでは慢性化しやすい [3] |
| **遺伝率** | 約40〜50% [4] |

| 用語 | 定義 | 例 |
|---|---|---|
| **強迫観念** | 繰り返し浮かぶ不快な考えやイメージ [2] | 「手が汚れている」「鍵を閉め忘れた」 |
| **強迫行為** | 不安を軽減するための反復的な行動 [2] | 何度も手を洗う、何度も確認する |

> **ポイント**
> - OCDは**不合理だとわかっていても止められない**のが特徴 [2]
> - 小児の**約1〜3%**に認められる [1]
> - こだわりとの違いは**苦痛の有無と生活への支障** [2]

---

### Q2.「子どもに多い強迫症状は？」

**お母さん**「子どもではどのような症状が多いですか？」

**おかもん先生**「子どもに多い強迫症状には特徴的なパターンがあります [3]。」

| 強迫観念のタイプ | 具体例 | 頻度 |
|---|---|---|
| **汚染への恐怖** | 「手にバイ菌がついている」 | 約40〜50% [3] |
| **攻撃的な考え** | 「誰かを傷つけてしまうかも」 | 約30% [3] |
| **対称性・正確さ** | 物が左右対称でないと不安 | 約30% [3] |
| **宗教的・道徳的** | 「悪いことを考えてしまった」 | 約10〜15% [3] |

| 強迫行為のタイプ | 具体例 | 頻度 |
|---|---|---|
| **洗浄・清潔** | 何度も手を洗う、入浴が長い | 約50% [3] |
| **確認** | 鍵、電気、ガスを何度も確認 | 約40% [3] |
| **配列・整頓** | 物を完璧に並べる | 約25% [3] |
| **数を数える** | 特定の回数繰り返さないと不安 | 約20% [3] |

> **ポイント**
> - **汚染恐怖＋手洗い**が子どもに最も多いパターン [3]
> - 子どもは強迫観念を言語化できないことがある
> - 複数のタイプが同時に存在することも多い

---

### Q3.「OCDと発達障害のこだわりの違いは？」

**お母さん**「ASDのこだわりとOCDの違いはなんですか？」

**おかもん先生**「両者は似て見えることがありますが、**本人の感じ方が大きく異なります** [5]。」

| 特徴 | OCD | ASDのこだわり |
|---|---|---|
| **本人の感じ方** | 苦痛、やめたいのにやめられない [2] | 心地よい、安心する [5] |
| **内容** | 不合理だと認識していることが多い | 本人にとって合理的 [5] |
| **機能** | 不安を軽減するため | 予測可能性・安心のため [5] |
| **介入への反応** | 行為を止められると強い不安 | ルーティンの変更で混乱 [5] |
| **経過** | 変動あり | 比較的安定 [5] |

「ただし、ASDとOCDは**併存することもある**ため、専門家による評価が重要です [5]。」

> **ポイント**
> - OCDは**苦痛を伴う**が、ASDのこだわりは**心地よい** [5]
> - OCDは**不安軽減のため**、ASDは**安心のため** [5]
> - 両者は**併存しうる**ため専門家の評価が必要

---

### Q4.「治療はどうすればよいですか？」

**お母さん**「OCDの治療法を教えてください」

**おかもん先生**「**曝露反応妨害法（ERP）を含む認知行動療法が第一選択**です [6]。」

| 治療法 | 内容 | エビデンス |
|---|---|---|
| **ERP（曝露反応妨害法）** | 強迫の対象にあえて向き合い、強迫行為をしない練習 [6] | 第一選択、約50〜70%が改善 |
| **認知行動療法（CBT）** | ERPを中核とした包括的アプローチ [6] | 最も強いエビデンス |
| **SSRI** | 中等症以上、ERP不応例に [3] | フルオキセチン、セルトラリン等 |
| **CBT＋SSRI** | 重症例に | 併用が最も効果的 [3] |
| **家族介入** | 家族の巻き込みへの対応 [6] | 治療効果を高める |

| ERPの具体例（手洗いの強迫の場合） |
|---|
| 1. 汚れていると感じるものに触れる（曝露） |
| 2. すぐに手を洗わずに我慢する（反応妨害） |
| 3. 不安が自然に下がることを体験する |
| 4. 段階的に難易度を上げていく |

> **ポイント**
> - **ERPが治療の柱**で、約50〜70%が改善 [6]
> - 薬物療法は中等症以上で検討 [3]
> - 家族が強迫行為に巻き込まれないことも重要

---

### Q5.「家族はどう対応すべきですか？」

**お母さん**「子どもの強迫行為に付き合うべきですか？」

**おかもん先生**「家族が強迫行為に**巻き込まれる（accommodation）**ことは、症状を悪化させます [6]。」

| 家族の巻き込みの例 | 推奨される対応 |
|---|---|
| 子どもの代わりに確認する | 確認行為に参加しない |
| 手洗いの回数を一緒に数える | 決められた回数で終わりにする |
| 強迫行為のための物品を用意する | 段階的に協力を減らす |
| 子どもの回避に合わせて生活を変える | 通常の生活を維持する |

| 家族が心がけること |
|---|
| OCDという病気を理解する |
| 本人を責めない（「なぜやめられないの」は禁句） |
| 強迫行為への巻き込みを段階的に減らす |
| 小さな進歩を認めて褒める |
| 専門家と連携して対応する |

> **ポイント**
> - 家族の**巻き込み（accommodation）**は症状を悪化させる [6]
> - 巻き込みを**段階的に減らす**ことが大切
> - 本人を責めず、**病気への理解**を深める
