# 愛育病院 小児科おかもん だより Vol.318

## 「うちの子の口、くさい？」――子どもの口臭の原因と対策

> **今号のポイント**
>
> 1. 子どもの口臭の原因の約90%は口腔内にある（舌苔・歯磨き不足・虫歯など）
> 2. 口呼吸による口腔乾燥は口臭の大きな原因。「お口ぽかん」は要注意
> 3. 朝の口臭は正常（睡眠中の唾液減少による）。持続する口臭は受診を検討

こんにちは。愛育病院小児科のおかもん先生です。

**「子どもの口がくさい気がします」「朝起きた時の口臭がひどいのですが、病気でしょうか？」**――口臭は、お子さん本人は気づきにくいけれど、保護者の方が心配されるテーマのひとつです。

「口が臭い＝内臓が悪い」と心配される方もいますが、**子どもの口臭の原因の約90%は口の中にあります** [1][2]。多くの場合、日常のケアで改善できます。今回は、子どもの口臭の原因と対策をお伝えします。

---

### Q1.「子どもの口臭の原因は何ですか？」

**お母さん**「3歳の子どもの口が最近臭います。何かの病気でしょうか？」

**おかもん先生**「子どもの口臭には**口の中の原因（口腔内要因）**と**口の外の原因（全身要因）**がありますが、**約90%は口腔内が原因**です [1][2]」

| 原因の分類 | 頻度 |
|---|---|
| **口腔内が原因** | **約90%** [1][2] |
| **耳鼻咽喉科的原因** | 約5〜8% [1] |
| **全身疾患が原因** | **非常にまれ**（1〜2%以下）[1] |

**おかもん先生**「口腔内の主な原因を詳しく見ていきましょう」

| 口腔内の原因 | 説明 | 頻度 |
|---|---|---|
| **舌苔（ぜったい）** | 舌の表面に細菌や食べかすが白〜黄色く付着したもの。口臭の**最大の原因** [1][2] | 最も多い |
| **歯磨き不足** | プラーク（歯垢）の蓄積により細菌が臭いの元（揮発性硫黄化合物）を産生 [2][3] | 多い |
| **虫歯（う蝕）** | 虫歯の穴に食べかすや細菌が溜まり臭いを発する [2] | やや多い |
| **口腔乾燥（ドライマウス）** | 唾液が減ると口腔内の自浄作用が低下し、細菌が増殖 [1][4] | 多い |
| **口呼吸** | 口の中が乾燥し、細菌が繁殖しやすくなる [4] | 多い |
| **歯肉炎** | 歯茎の炎症。歯磨き不足が原因のことが多い [2] | やや多い |

**お母さん**「舌苔が一番の原因なんですか？」

**おかもん先生**「はい。成人の口臭研究でも小児の口臭研究でも、**舌苔の付着量が口臭の強さと最も強く相関する**ことがわかっています [1][2]。舌の表面には無数の乳頭（小さな突起）があり、そこに細菌・食べかす・剥がれた粘膜細胞が溜まります。これらが分解される際に、**揮発性硫黄化合物（VSC）**という臭い物質が発生するのです [1][3]」

**おかもん先生**「口腔外の原因も知っておきましょう」

| 口腔外の原因 | 説明 |
|---|---|
| **副鼻腔炎（蓄膿症）** | 後鼻漏（鼻水が喉に流れる）により、喉の奥で細菌が繁殖 [5] |
| **扁桃の膿栓（のうせん）** | 扁桃のくぼみに食べかすや細菌が溜まった白い塊。悪臭を放つ [6] |
| **アデノイド肥大** | 鼻呼吸障害→口呼吸→口腔乾燥→口臭 [4] |
| **鼻腔内異物** | 幼児が鼻に小さなおもちゃなどを入れてしまい、片側の鼻から悪臭のある鼻汁 [5] |
| **全身疾患** | 糖尿病（アセトン臭）、肝疾患、腎疾患など。**小児では非常にまれ** [1] |

> **ポイント**
> - 子どもの口臭の**約90%は口の中が原因** [1][2]
> - **舌苔が最大の原因**。揮発性硫黄化合物（VSC）が臭いの正体 [1][3]
> - **全身疾患による口臭は小児では非常にまれ** [1]

### Q2.「朝の口臭がひどいのですが、正常ですか？」

**お母さん**「朝起きた時の口臭がとても気になります。病気のサインですか？」

**おかもん先生**「**朝の口臭は正常です**。これは**生理的口臭**と呼ばれるもので、病気ではありません [1][2]」

| 生理的口臭が起こる理由 | 説明 |
|---|---|
| **睡眠中の唾液分泌低下** | 寝ている間は唾液の分泌量が大幅に減少する [1][4] |
| **唾液の自浄作用の低下** | 唾液には口の中を洗い流す作用がある。分泌が減ると細菌が増殖 [4] |
| **細菌の増殖** | 夜間に口腔内細菌が急増し、VSC（臭い物質）の産生が増える [1] |
| **口呼吸** | 睡眠中に口が開いていると口腔乾燥が加速し、口臭が悪化 [4] |

**おかもん先生**「**朝食を食べて、歯を磨いて、水分を摂れば改善する口臭は生理的口臭**であり、心配ありません [1][2]。問題なのは、日中を通して持続する口臭や、歯磨きをしても改善しない口臭です」

| 口臭のタイプ | 心配度 |
|---|---|
| 朝だけ臭い → 食事・歯磨き後に消える | **正常（生理的口臭）** [1] |
| 空腹時に臭い → 食事後に消える | **正常** |
| 日中も持続する口臭 | **原因の検索が必要** [2] |
| 歯磨きをしても改善しない | **受診を検討** [2] |
| 片側の鼻だけ臭い | **鼻腔内異物の可能性→耳鼻科受診** [5] |

> **ポイント**
> - **朝の口臭は正常**（生理的口臭）。唾液の減少が原因 [1][4]
> - 食事・歯磨き後に消える口臭は心配不要 [1][2]
> - **日中も持続する口臭**は原因の検索が必要 [2]

### Q3.「口呼吸が口臭の原因になるんですか？」

**お母さん**「うちの子はいつも口が開いています。それが口臭の原因ですか？」

**おかもん先生**「はい。**口呼吸は子どもの口臭の大きな原因のひとつ**です [4][7]。口呼吸を続けると口の中が乾燥し、唾液による自浄作用が低下します。その結果、細菌が増殖して口臭が発生します」

| 口呼吸が口臭を起こすメカニズム | 説明 |
|---|---|
| **口腔乾燥** | 常に空気が口の中を通るため、唾液が蒸発して口が乾く [4][7] |
| **唾液の自浄作用低下** | 唾液には細菌を洗い流す・抗菌物質を供給する役割がある [4] |
| **細菌の増殖** | 乾燥環境では嫌気性菌（酸素が少ない環境を好む菌）が増え、VSCを多く産生 [1] |
| **舌苔の増加** | 口腔乾燥により舌の表面に汚れが蓄積しやすくなる [1] |

**おかもん先生**「**口呼吸の原因**は以下のようなものがあります」

| 口呼吸の原因 | 対応 |
|---|---|
| **アレルギー性鼻炎** | 鼻詰まりの治療（点鼻薬、抗アレルギー薬）[7] |
| **アデノイド肥大** | 耳鼻咽喉科で評価。高度な場合は手術を検討 [7] |
| **鼻中隔弯曲** | 成長とともに改善することもある。高度な場合は手術 [7] |
| **習慣性口呼吸** | 鼻の問題がないのに口で呼吸する癖。意識的な鼻呼吸の練習 [7] |
| **口腔周囲筋の発達不全** | 「あいうべ体操」などの口腔筋機能療法 [7] |

**おかもん先生**「口呼吸の改善は口臭だけでなく、**歯並び・顔面の成長・睡眠の質**にも良い影響があります [7]。お口ぽかんが気になる場合は、まず鼻の通りに問題がないか耳鼻科で確認し、必要に応じて治療を受けてください」

> **ポイント**
> - **口呼吸は口臭の大きな原因**。口腔乾燥により細菌が増殖する [4][7]
> - 口呼吸の原因（鼻炎・アデノイドなど）を治療することが根本的対策 [7]
> - 口呼吸の改善は口臭だけでなく**歯並び・睡眠の質**にも好影響 [7]

### Q4.「口臭を改善する方法と、受診の目安を教えてください」

**お母さん**「家庭でできる口臭対策を教えてください。どんな時に受診すべきですか？」

**おかもん先生**「家庭でできる対策をまとめます」

| 対策 | 具体的方法 |
|---|---|
| **舌磨き** | やわらかい歯ブラシまたは舌クリーナーで、舌の奥から手前に向かって**やさしく**数回なでる。1日1回で十分 [1][2] |
| **丁寧な歯磨き** | 仕上げ磨きを確実に行う。歯と歯茎の境目、歯の裏側を意識 [2][3] |
| **水分摂取** | こまめに水やお茶を飲ませて口腔内の乾燥を防ぐ [4] |
| **鼻呼吸の促進** | 日中に口を閉じるよう意識させる。「あいうべ体操」も有効 [7] |
| **規則正しい食事** | よく噛んで食べることで唾液分泌が促進される [4] |
| **定期的な歯科健診** | 虫歯・歯肉炎の早期発見と治療 [2] |

**おかもん先生**「**舌磨きのコツ**をもう少し詳しくお伝えします」

| 舌磨きの注意点 | 説明 |
|---|---|
| **力を入れすぎない** | 舌の表面を傷つけると逆効果 [1] |
| **奥から手前へ** | 嘔吐反射を避けるため、奥から手前方向に動かす [1] |
| **1日1回** | 朝の歯磨き時に行うのが効果的 [1] |
| **専用ツール** | 舌クリーナーが理想だが、やわらかい歯ブラシでもOK [1] |

**おかもん先生**「受診の目安もまとめます」

| 受診の目安 | 受診先 |
|---|---|
| 歯磨き・舌磨きをしても口臭が持続する | **歯科**（虫歯・歯肉炎の確認）[2] |
| 鼻づまりが続いている、黄色〜緑の鼻水 | **耳鼻咽喉科**（副鼻腔炎の可能性）[5] |
| 片側の鼻だけ臭い | **耳鼻咽喉科**（鼻腔内異物の可能性。**緊急度高い**）[5] |
| いびきがひどい、睡眠中に口が常に開いている | **耳鼻咽喉科**（アデノイド肥大の可能性）[7] |
| 扁桃腺が大きい、白い塊が見える | **耳鼻咽喉科**（膿栓の可能性）[6] |
| 口臭以外に全身症状（体重減少、多飲多尿など）がある | **小児科**（全身疾患の除外）[1] |

> **ポイント**
> - **舌磨き**が最も効果的な口臭対策。1日1回、やさしく [1][2]
> - **水分摂取**と**鼻呼吸の促進**で口腔乾燥を防ぐ [4][7]
> - 持続する口臭は**歯科**、鼻症状があれば**耳鼻咽喉科**を受診 [2][5]

## まとめ

- 子どもの口臭の**約90%は口腔内が原因**（舌苔・歯磨き不足・虫歯）[1][2]
- **朝の口臭は正常**（生理的口臭）。食事・歯磨きで消えれば心配不要 [1]
- **口呼吸による口腔乾燥**は口臭の大きな原因。鼻呼吸の促進が重要 [4][7]
- **舌磨き**が最も効果的。1日1回、奥から手前にやさしくなでる [1][2]
- **全身疾患による口臭は小児では非常にまれ** [1]
- 日中も持続する口臭、片側だけの臭い鼻水がある場合は**受診を** [2][5]
