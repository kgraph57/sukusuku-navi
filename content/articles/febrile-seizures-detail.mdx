# 愛育病院 小児科**おかもん先生** だより Vol.251

## 「熱が出てけいれんしました！」――熱性けいれん

こんにちは。愛育病院小児科の**おかもん先生**です。

熱性けいれんは、乳幼児が発熱時にけいれんを起こす疾患で、小児の約8%が経験します。保護者にとって非常に怖い経験ですが、多くは予後良好です。今回は、熱性けいれんについてお伝えします。

---

### Q1.「熱性けいれんとは？」

**お母さん**「熱が出てガクガクしました。てんかんですか？」

**おかもん先生**「熱性けいれんは**発熱に伴って起こるけいれん**で、てんかんとは異なります [1]。」

| 基本情報 | 詳細 |
|---|---|
| **定義** | 38℃以上の発熱に伴うけいれん |
| **好発年齢** | 生後6か月〜5歳（ピークは1-2歳） |
| **頻度** | 小児の約8%（日本は欧米より高い） |
| **遺伝** | 家族歴がある場合が多い |

| 単純型と複雑型 |
|---|
| **単純型**（約70%）：15分以内、全身性、24時間以内に1回のみ |
| **複雑型**（約30%）：15分以上、または部分的、または24時間に複数回 |

> **ポイント**
> - 日本の小児の約8%が経験
> - てんかんとは異なる
> - 単純型は予後良好

---

### Q2.「けいれんが起きたらどうすればいい？」

**お母さん**「けいれんが起きた時、何をすればいいですか？」

**おかもん先生**「**慌てず安全を確保**することが最も大切です [2]。」

| すべきこと | 詳細 |
|---|---|
| **横向きに寝かせる** | 嘔吐物による窒息予防 |
| **時間を計る** | けいれんの持続時間を確認 |
| **安全確保** | 周囲の危険物を除去 |
| **衣服をゆるめる** | 呼吸を楽にする |
| **観察** | けいれんの様子を観察（動画撮影も有用） |

| してはいけないこと |
|---|
| 口に物を入れない（指も入れない） |
| 体を押さえつけない |
| 大声で呼びかけない |
| 水を飲ませない |

「**動画を撮影**しておくと、医師の診断に非常に役立ちます。」

> **ポイント**
> - 横向きに寝かせる
> - 口に物を入れない
> - 時間を計り、動画を撮影

---

### Q3.「救急車を呼ぶべき？」

**お母さん**「救急車を呼ぶ基準はありますか？」

**おかもん先生**「以下の場合は**119番に電話**してください [3]。」

| 救急車を呼ぶべき場合 | 詳細 |
|---|---|
| **5分以上続く** | けいれん重積の可能性 |
| **意識が戻らない** | けいれん後30分以上ボーッとしている |
| **初めてのけいれん** | 原因の確認が必要 |
| **繰り返す** | 24時間以内に複数回 |
| **左右差がある** | 片側だけのけいれん |
| **生後6か月未満** | 髄膜炎の除外が必要 |

「**初めてのけいれんは必ず受診**してください。熱性けいれん以外の原因を除外する必要があります。」

> **ポイント**
> - 5分以上続けば救急車
> - 初めてのけいれんは必ず受診
> - 片側だけの場合も要注意

---

### Q4.「再発予防は必要ですか？」

**お母さん**「また起きるのが心配です。予防薬はありますか？」

**おかもん先生**「予防薬は**全員に必要なわけではありません** [4]。」

| 再発率 | 詳細 |
|---|---|
| **初回後** | 約30-40%が再発 |
| **2回目後** | 約50%が再発 |
| **リスク因子** | 1歳未満の初発、家族歴、低い体温で発症 |

| ジアゼパム坐薬（ダイアップ）の使用基準 |
|---|
| 遷延性（15分以上）のけいれんの既往 |
| けいれんの反復（短期間に複数回） |
| 医師と相談して個別に判断 |

「**解熱剤で熱性けいれんを予防することはできません**。解熱剤は痛みや不快感を和らげるために使用します。」

> **ポイント**
> - 再発率は約30-40%
> - 予防薬は全員に必要ではない
> - 解熱剤では予防できない

---

### Q5.「てんかんになりますか？」

**お母さん**「将来てんかんになりますか？知能に影響はありますか？」

**おかもん先生**「**単純型熱性けいれんの予後は非常に良好**です [5]。」

| 予後 | 詳細 |
|---|---|
| **知能への影響** | 単純型では影響なし |
| **てんかん移行率** | 一般（1%）より少し高い（2-4%） |
| **自然消失** | 5歳を過ぎると起こらなくなる |

| てんかんリスクが高い場合 |
|---|
| 複雑型熱性けいれん |
| 発達の遅れがある |
| てんかんの家族歴がある |

「**単純型熱性けいれんは脳に障害を残しません**。安心してください。」

> **ポイント**
> - 単純型は知能に影響なし
> - 5歳を過ぎると起こらなくなる
> - てんかんへの移行は少ない

---

## 今号のまとめ

- **熱性けいれんは小児の約8%が経験**
- **横向きに寝かせ、口に物を入れない**
- **5分以上続けば救急車**
- **解熱剤では予防できない**
- **単純型の予後は非常に良好**
