# 愛育病院 小児科**おかもん先生** だより Vol.203

## 「どんな咳が危ないの？」――小児の咳の見分け方

こんにちは。愛育病院小児科の**おかもん先生**です。

子どもの咳は小児科を受診する理由として最も多い症状の一つです。咳の種類（音、タイミング、持続期間）によって原因がある程度推定できます。今回は、保護者が知っておくべき「咳の見分け方」をお伝えします。

---

### Q1.「咳の種類で原因が分かりますか？」

**お母さん**「咳にもいろいろあるのですか？」

**おかもん先生**「咳の音やパターンで、原因をある程度推定できます [1]。」

| 咳の特徴 | 疑われる原因 |
|---|---|
| **犬吠様咳（ケンケン）** | クループ症候群 |
| **連続的な咳込み＋ヒューという吸気音** | 百日咳 |
| **夜間のゼーゼー＋咳** | 喘息 |
| **湿った咳（痰がらみ）** | 気管支炎、肺炎 |
| **食事中・直後の咳** | 誤嚥、胃食道逆流 |
| **起床時の咳** | 副鼻腔炎（後鼻漏） |
| **突然始まった激しい咳** | 気道異物 |

> **ポイント**
> - 咳の音とタイミングが診断のヒント
> - 犬吠様＝クループ、ゼーゼー＝喘息
> - 突然の激しい咳は気道異物を疑う

---

### Q2.「すぐに受診すべき咳は？」

**お母さん**「どんな咳なら急いで受診すべきですか？」

**おかもん先生**「以下の咳は**緊急性が高い**です [2]。」

| 緊急受診すべき咳 | 理由 |
|---|---|
| **呼吸困難を伴う咳** | 肺炎、喘息大発作、気道異物 |
| **チアノーゼ（唇が青い）＋咳** | 重度の低酸素 |
| **突然始まった激しい咳（異物を疑う）** | 窒息のリスク |
| **犬吠様咳＋吸気性喘鳴** | 重症クループ |
| **血痰を伴う咳** | 肺出血、結核等 |

| 数日以内に受診すべき咳 |
|---|
| 3週間以上続く咳 |
| 夜間に目が覚めるほどの咳 |
| 発熱が3日以上続く＋咳 |
| 食欲が著しく低下 |

> **ポイント**
> - 呼吸困難＋咳は緊急
> - 突然の激しい咳は気道異物を疑う
> - 3週間以上続く咳は要受診

---

### Q3.「夜の咳がひどい場合は？」

**お母さん**「夜になると咳が悪化します」

**おかもん先生**「夜間の咳には複数の原因があります [3]。」

| 原因 | 特徴 | 対策 |
|---|---|---|
| **後鼻漏** | 鼻水がのどに落ちて咳 | 鼻吸引、頭を少し高く |
| **喘息** | ゼーゼーを伴う | かかりつけ医で喘息の評価 |
| **胃食道逆流** | 食後に悪化 | 就寝前の食事を避ける |
| **クループ** | 犬吠様咳 | 冷たい空気、受診 |
| **乾燥** | 冬場に悪化 | 加湿器で湿度50-60% |

「**鼻水が原因の咳**は非常に多いです。寝る前に鼻吸引をするだけで、夜の咳が大幅に改善することがあります。」

> **ポイント**
> - 夜の咳の原因は複数ある
> - 鼻吸引が効果的なことが多い
> - ゼーゼーを伴う場合は喘息の評価を

---

### Q4.「気道異物はどう見分けますか？」

**お母さん**「何かを誤飲した可能性がある場合は？」

**おかもん先生**「**突然始まった激しい咳**は気道異物を強く疑います [4]。」

| 気道異物のサイン | 特徴 |
|---|---|
| **突然の激しい咳** | それまで元気だったのに突然 |
| **喘鳴** | 片側性のゼーゼー |
| **声が出ない** | 完全閉塞のサイン（緊急） |
| **好発年齢** | 1-3歳 |
| **多い異物** | ピーナッツ、枝豆、小さなおもちゃ |

| 窒息時の対応 |
|---|
| **1歳未満**：背部叩打法（うつ伏せにして背中を5回叩く） |
| **1歳以上**：ハイムリック法（腹部突き上げ法） |
| **意識がない**：CPR（心肺蘇生）を開始し119番 |

「**3歳未満の子どもにはピーナッツ・ナッツ類を与えない**でください。気道異物の原因として最も多い食品です。」

> **ポイント**
> - 突然の激しい咳は気道異物を疑う
> - 3歳未満にナッツ類は与えない
> - 窒息時の対応（背部叩打法・ハイムリック法）を覚えておく

---

### Q5.「咳を楽にする方法は？」

**お母さん**「咳が出ている時、何をしてあげられますか？」

**おかもん先生**「年齢に応じた対策があります [5]。」

| 対策 | 年齢 | 方法 |
|---|---|---|
| **鼻吸引** | 全年齢 | 鼻水を吸引して後鼻漏を防ぐ |
| **水分補給** | 全年齢 | こまめに水分を与える |
| **加湿** | 全年齢 | 部屋の湿度を50-60%に |
| **頭を少し高く** | 全年齢 | タオルで枕を少し高くする |
| **蜂蜜** | 1歳以上 | ティースプーン1杯。咳を軽減 |
| **のど飴** | 4歳以上 | のどの保湿効果 |

「**市販の咳止め薬は6歳未満には推奨されません**。効果のエビデンスが乏しく、副作用のリスクがあるためです。」

> **ポイント**
> - 鼻吸引と水分補給が基本
> - 蜂蜜は1歳以上で効果あり
> - 市販の咳止めは6歳未満に推奨されない

---

## 今号のまとめ

- **咳の音とタイミングで原因を推定**できます
- **呼吸困難を伴う咳は緊急**。突然の激しい咳は気道異物
- **3週間以上続く咳**は受診
- **鼻吸引・水分・加湿・蜂蜜（1歳以上）**が家庭での対策
- **市販の咳止めは6歳未満には推奨されない**
