# 愛育病院 小児科**おかもん先生** だより Vol.161

## 「パンもうどんもダメ？」——小麦アレルギーの管理と代替食品

> **今号のポイント**
>
> 1. 小麦アレルギーは卵・牛乳に次いで3番目に多く、約80%が学童期までに耐性を獲得する
> 2. 小麦は主食に使われるため除去の影響が大きいが、米粉やグルテンフリー食品で代替可能
> 3. 小麦依存性運動誘発アナフィラキシー（WDEIA）は学童期以降に多い特殊な病型

こんにちは。愛育病院小児科の**おかもん先生**先生です。

**「小麦アレルギーと診断されましたが、パンもうどんもダメですか？」「毎日の食事をどうしたらいいですか？」**——小麦は日本人の食生活に深く根付いた食材だけに、除去指示を受けた時の戸惑いはとても大きいと思います。

しかし、**多くのお子さんは成長とともに食べられるようになります** [1]。今回は、小麦アレルギーの管理と代替食品について詳しくお伝えします。

---

### Q1.「小麦アレルギーの頻度と原因について教えてください」

**お母さん**「小麦アレルギーはどのくらいいますか？」

**おかもん先生**「**食物アレルギーの原因食物として卵・牛乳に次いで3番目に多い**です [1]」

**小麦アレルギーの基本情報 [1][2]:**

| 項目 | 内容 |
|---|---|
| **頻度** | 乳幼児の食物アレルギーの約8〜10% |
| **主要アレルゲン** | **ω-5グリアジン**が最も重要 [2] |
| **その他のアレルゲン** | グルテン（グリアジン、グルテニン）、水溶性タンパク |
| **耐性獲得** | **約80%が学童期までに耐性を獲得** [1] |
| **交差反応** | 大麦、ライ麦と一部交差反応あり [2] |

**おかもん先生**「重要なのは**小麦アレルギーとセリアック病は異なる疾患**ということです [2]。セリアック病はグルテンに対する自己免疫疾患で、日本人にはまれです。小麦アレルギーはIgEを介したアレルギー反応が主体です」

| 比較 | 小麦アレルギー | セリアック病 |
|---|---|---|
| **機序** | IgE依存性アレルギー | 自己免疫疾患 |
| **頻度（日本）** | 比較的多い | まれ |
| **耐性獲得** | 多くが獲得する | 生涯グルテン除去が必要 |
| **検査** | IgE抗体、負荷試験 | 抗tTG抗体、小腸生検 |

> **ポイント**
> - 小麦は食物アレルギーの**3番目に多い**原因 [1]
> - **約80%**が学童期までに耐性獲得 [1]
> - **小麦アレルギーとセリアック病は別疾患** [2]

---

### Q2.「小麦を除去した場合、何を食べさせればいいですか？」

**お母さん**「パンもうどんも食べられないと、主食が困ります」

**おかもん先生**「**米が主食の日本では、比較的代替がしやすい**です [3]。また、最近は米粉製品やグルテンフリー食品の選択肢が増えています」

**主な代替食品 [3]:**

| 除去する食品 | 代替食品 |
|---|---|
| **パン** | **米粉パン**、米粉蒸しパン |
| **うどん** | **ビーフン**、フォー、春雨、**米粉うどん** |
| **パスタ** | **米粉パスタ**、グルテンフリーパスタ |
| **小麦粉（料理用）** | **米粉、片栗粉、タピオカ粉** |
| **醤油** | **小麦不使用の醤油**（たまり醤油など） |
| **揚げ物の衣** | **米粉、コーンフレーク粉砕** |
| **ケーキ・クッキー** | 米粉スイーツ、芋ようかん |

**おかもん先生**「**醤油**については、製造過程で小麦タンパクが分解されるため、**多くの小麦アレルギーのお子さんは普通の醤油を使用可能**です [3]。ただし重症例では注意が必要ですので、主治医に確認してください」

> **ポイント**
> - **米が主食の日本では代替がしやすい** [3]
> - **米粉製品**の選択肢が充実している [3]
> - **醤油は多くの場合使用可能**だが、重症例は主治医に確認 [3]

---

### Q3.「大麦やライ麦も避けるべきですか？」

**お母さん**「麦茶は飲めますか？大麦もダメですか？」

**おかもん先生**「小麦と他の麦類の**交差反応**について整理しましょう [4]」

**小麦と他の穀物の関係 [4]:**

| 穀物 | 小麦との交差反応 | 実際に症状が出る頻度 |
|---|---|---|
| **大麦** | タンパク質の相同性あり | **約20%**が反応する可能性 |
| **ライ麦** | タンパク質の相同性あり | **約20%**が反応する可能性 |
| **オート麦** | 相同性はやや低い | 多くは摂取可能 |
| **米** | 相同性なし | **問題なし** |
| **とうもろこし** | 相同性なし | **問題なし** |
| **そば** | 相同性なし | **問題なし**（別のアレルギーに注意） |

**おかもん先生**「**麦茶**は大麦から作られますが、タンパク質の含有量がごく微量のため、**多くの小麦アレルギーのお子さんは問題なく飲める**ことが多いです [4]。ただし個人差がありますので、初めて飲む場合は少量から試してください」

> **ポイント**
> - 大麦・ライ麦は**約20%**が反応する可能性 [4]
> - **麦茶**は多くの場合問題なく飲める [4]
> - **米・とうもろこし**は交差反応なし [4]

### Q4.「小麦依存性運動誘発アナフィラキシーについて教えてください」

**お母さん**「給食でパンを食べた後、体育で具合が悪くなったと聞きましたが…」

**おかもん先生**「**小麦依存性運動誘発アナフィラキシー（WDEIA）**は、**小麦を食べた後に運動するとアナフィラキシーが起きる**特殊な病型です [5]」

**WDEIAの特徴 [5][6]:**

| 項目 | 内容 |
|---|---|
| **好発年齢** | **学童期〜成人**（幼児はまれ） |
| **主要アレルゲン** | **ω-5グリアジン** [5] |
| **発症条件** | 小麦摂取後**2〜4時間以内の運動** |
| **増悪因子** | 運動、NSAIDs（痛み止め）、アルコール、入浴 [6] |
| **症状** | 蕁麻疹→呼吸困難→血圧低下→アナフィラキシー |

**対策 [5][6]:**

| 対策 | 内容 |
|---|---|
| **食後の運動制限** | 小麦摂取後**最低2時間**は激しい運動を避ける |
| **エピペン処方** | 診断された場合は常に携帯 |
| **学校への情報提供** | 給食後の体育・部活動について配慮を依頼 |
| **食前の運動** | 食前の運動は問題なし |

**おかもん先生**「WDEIAは**食べるだけなら症状が出ず、運動が加わると発症**するのが特徴です [5]。学校生活での管理が重要で、給食・体育の時間配慮が必要になることがあります」

> **ポイント**
> - **小麦＋運動**でアナフィラキシーを起こす特殊な病型 [5]
> - 小麦摂取後**最低2時間**は激しい運動を避ける [5][6]
> - **エピペン**の携帯と学校への情報提供が重要 [6]

---

### Q5.「食品表示の注意点と管理のコツを教えてください」

**お母さん**「小麦が含まれていることに気づかない食品はありますか？」

**おかもん先生**「小麦は**特定原材料として表示義務**がありますが、意外な食品に含まれていることがあります [7]」

**見落としやすい小麦含有食品 [7]:**

| 食品 | 小麦の使われ方 |
|---|---|
| **醤油** | 原料の一つ（多くの場合は摂取可能） |
| **カレールウ** | 小麦粉がベース |
| **フライ・天ぷら** | 衣に小麦粉・パン粉使用 |
| **ハンバーグ** | つなぎに小麦粉・パン粉使用 |
| **ミートボール** | つなぎに小麦粉使用 |
| **麩（ふ）** | グルテンそのもの |
| **餃子の皮** | 小麦粉が主原料 |

**管理のコツ [7][8]:**

| コツ | 内容 |
|---|---|
| **米粉レシピの活用** | 米粉のレパートリーを増やすと食事が楽しくなる |
| **グルテンフリー表示** | 海外製品のGFマークも参考に |
| **外食時** | アレルギー対応メニューのあるお店を事前に確認 |
| **栄養バランス** | 小麦除去でもカロリー・栄養は米や他の穀物で十分補える |

> **ポイント**
> - 小麦は**特定原材料として表示義務あり** [7]
> - カレールウ、フライ、ハンバーグなど**意外な食品に注意** [7]
> - **米粉レシピ**のレパートリーを増やすと管理が楽に [8]

## まとめ

- 小麦アレルギーは食物アレルギーの**3番目に多い**原因。**約80%**が学童期までに耐性獲得 [1]
- **米が主食の日本では代替がしやすい**。米粉製品が充実 [3]
- **醤油は多くの場合使用可能**。麦茶も多くは問題なし [3][4]
- **WDEIA**は小麦＋運動でアナフィラキシーが起きる特殊型 [5]
- 意外な食品に小麦が含まれているので**表示を毎回確認** [7]
