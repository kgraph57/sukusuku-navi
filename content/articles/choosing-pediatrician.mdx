# 愛育病院 小児科**おかもん先生** だより Vol.295

## 「この先生でいいの？」――かかりつけ医の選び方

こんにちは。愛育病院小児科の**おかもん先生**です。

「小児科はどこに行けばいいですか？」「かかりつけ医はどうやって選べばいいですか？」――お子さんの健康を守るパートナーとしての**かかりつけ小児科医**を見つけることは、子育てにおいて非常に重要です [1]。

---

### Q1.「かかりつけ医とは？」

**お母さん**「かかりつけ医って何ですか？ 普段行く病院のことですか？」

**おかもん先生**「かかりつけ医とは、**お子さんの健康について何でも相談でき、日常的な医療を提供してくれる身近な医師**のことです [1]。」

| 項目 | かかりつけ医の役割 |
|---|---|
| **日常の診療** | 風邪、発熱、湿疹等の一般的な疾患 [1] |
| **予防接種** | スケジュール管理と接種 |
| **乳幼児健診** | 発育・発達の経過観察 |
| **健康相談** | 育児の悩み、発達の相談 |
| **専門医への紹介** | 必要な場合に適切な専門医を紹介 [1] |
| **慢性疾患の管理** | アレルギー、喘息等の長期管理 |

| かかりつけ医を持つメリット |
|---|
| お子さんの**成長の経過を把握**してもらえる |
| **予防接種の打ち忘れ防止** |
| 些細なことでも**気軽に相談**できる |
| **いつもと違う状態**に気づいてもらいやすい |
| 専門医受診が必要な場合に**スムーズに紹介**してもらえる |

> **ポイント**
> - かかりつけ医は**健康のパートナー** [1]
> - お子さんの成長を**継続的に見守ってもらえる**
> - 複数のかかりつけ医を持つよりも**一か所に集約**が理想

---

### Q2.「かかりつけ医を選ぶポイントは？」

**お母さん**「どのように選べばよいですか？」

**おかもん先生**「以下のポイントを参考にしてください [2]。」

| 選ぶポイント | チェック項目 |
|---|---|
| **自宅からの距離** | 体調が悪い時に通える距離か |
| **診療時間** | 仕事との両立ができる時間帯か |
| **予約システム** | オンライン予約、待ち時間の目安 |
| **医師との相性** | 質問しやすい雰囲気か |
| **説明のわかりやすさ** | 病気や治療について丁寧に説明してくれるか |
| **予防接種の対応** | 同時接種に対応しているか |
| **急患対応** | 急な体調不良時の対応方針 |

| 良いかかりつけ医の特徴 |
|---|
| 質問に対して**丁寧に説明**してくれる |
| **不安を受け止めてくれる** |
| 必要に応じて**専門医を紹介**してくれる |
| **過剰な検査や投薬をしない** |
| 予防接種の**スケジュールを管理**してくれる |
| 保護者の話を**よく聴いてくれる** |

> **ポイント**
> - **距離と診療時間**は実用的に重要 [2]
> - **医師との相性**（相談しやすさ）が最も大切
> - 引っ越し後やお子さん誕生後は**早めに見つける**

---

### Q3.「かかりつけ医と合わないと感じたら？」

**お母さん**「今のかかりつけ医と合わない気がします。変えてもいいですか？」

**おかもん先生**「**かかりつけ医は変えてもかまいません** [2]。大切なのは保護者とお子さんが安心して通えることです。」

| かかりつけ医を変えることを検討すべきサイン |
|---|
| 質問しても**十分に説明してもらえない** |
| **不安や心配を軽く扱われる**と感じる |
| 子どもが**極端に怖がる** |
| **待ち時間が長すぎて**通院が困難 |
| **予防接種のスケジュール管理**をしてくれない |

| かかりつけ医を変える時の注意 |
|---|
| 予防接種記録、既往歴を新しい医師に伝える |
| 母子手帳を持参する |
| 慢性疾患がある場合は紹介状をもらう |
| 前の医師の悪口は言わない |

> **ポイント**
> - かかりつけ医は**自由に変更できる** [2]
> - 変更時は**予防接種記録と既往歴**を新医師に伝える
> - 保護者とお子さんが**安心して通える**ことが最優先

---

### Q4.「セカンドオピニオンは失礼ですか？」

**お母さん**「他の先生の意見も聞いてみたいのですが、失礼ではないですか？」

**おかもん先生**「**セカンドオピニオンは患者の権利**であり、全く失礼ではありません [3]。」

| セカンドオピニオンが有用な場面 |
|---|
| 診断に**納得がいかない**場合 |
| 治療方針に**不安がある**場合 |
| **手術や長期治療**を提案された場合 |
| 症状が**改善しない**場合 |

| セカンドオピニオンの受け方 |
|---|
| かかりつけ医に「他の先生の意見も聞いてみたい」と伝える |
| 紹介状（診療情報提供書）をもらう |
| 検査結果のコピーをもらう |
| セカンドオピニオン外来がある病院を選ぶ |

> **ポイント**
> - セカンドオピニオンは**患者の権利** [3]
> - かかりつけ医に率直に伝えてOK
> - **紹介状と検査結果**があると効率的

---

### Q5.「港区でかかりつけ医を見つけるには？」

**お母さん**「港区で小児科を探すにはどうすればよいですか？」

**おかもん先生**「いくつかの方法で探せます [4]。」

| 探し方 | 方法 |
|---|---|
| **港区医師会のウェブサイト** | 港区内の医療機関一覧 [4] |
| **港区公式サイト** | 地域別の医療機関情報 |
| **東京都医療機関案内サービス（ひまわり）** | 診療科目、時間帯で検索 [4] |
| **口コミ** | ママ友、子育てひろばでの情報 |
| **保健所の保健師** | 地域の医療機関に詳しい |

| かかりつけ医デビューのタイミング |
|---|
| **出産前**に小児科を調べておく |
| **1か月健診後**に予防接種で初受診 |
| **引越し後**は早めに見つける |
| **お子さんが元気な時**に一度受診しておく |

> **ポイント**
> - **出産前から小児科を調べておく**と安心
> - 予防接種が最初の受診のきっかけになることが多い
> - **複数の情報源**を活用して探す [4]
