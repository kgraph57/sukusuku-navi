# 愛育病院 小児科**おかもん先生** だより Vol.265

## 「おしりが赤くなっています」――おむつかぶれ

こんにちは。愛育病院小児科の**おかもん先生**です。

おむつかぶれ（おむつ皮膚炎）は、ほとんどの赤ちゃんが経験する身近なトラブルです。適切なスキンケアで予防・改善できます。今回は、おむつかぶれについてお伝えします。

---

### Q1.「おむつかぶれの原因は？」

**お母さん**「なぜおしりが赤くなるのですか？」

**おかもん先生**「おむつかぶれは**おむつ内の湿気と刺激**が原因です [1]。」

| 原因 | 詳細 |
|---|---|
| **湿気** | おむつ内の蒸れ |
| **尿・便の刺激** | 特に下痢便は刺激が強い |
| **摩擦** | おむつと皮膚の擦れ |
| **長時間の着用** | 交換頻度が少ない |

| おむつかぶれの症状 |
|---|
| おしりの赤み（おむつが当たる部分） |
| 皮膚のただれ |
| 機嫌が悪い（おむつ交換時に泣く） |

> **ポイント**
> - 湿気と便の刺激が主な原因
> - 下痢の時に悪化しやすい
> - こまめなおむつ交換が予防の基本

---

### Q2.「予防と対処法は？」

**お母さん**「おむつかぶれの予防法は？」

**おかもん先生**「**こまめなおむつ交換と適切なスキンケア**が基本です [2]。」

| 予防・対処法 | 内容 |
|---|---|
| **こまめに交換** | 1-2時間ごと、便の後すぐに |
| **やさしく洗う** | ぬるま湯で洗い流す（おしりふきより有効） |
| **十分に乾かす** | 完全に乾いてからおむつを当てる |
| **保護クリーム** | ワセリンや亜鉛華軟膏でバリア |
| **おむつフリー** | 可能なら時々おむつを外す |

| おしりの拭き方 |
|---|
| 擦らず、押さえるように拭く |
| 座浴（お湯で洗い流す）が最も効果的 |
| ペットボトルシャワーも便利 |

> **ポイント**
> - お湯で洗い流すのが最も効果的
> - ワセリンでバリアを作る
> - 時々おむつを外して乾かす

---

### Q3.「カンジダとの見分け方は？」

**お母さん**「なかなか治りません」

**おかもん先生**「治りにくい場合は**カンジダ皮膚炎**の可能性があります [3]。」

| 特徴 | おむつかぶれ | カンジダ皮膚炎 |
|---|---|---|
| **部位** | おむつが当たる部分 | しわの奥にも広がる |
| **見た目** | 赤み、ただれ | 鮮紅色、辺縁に小さな衛星病変 |
| **治療** | 保湿・保護 | 抗真菌薬が必要 |
| **ステロイド** | 改善する | かえって悪化する |

「1週間以上改善しない場合は**カンジダの可能性**がありますので、受診してください。」

> **ポイント**
> - 1週間以上治らなければ受診
> - カンジダはしわの奥にも広がる
> - ステロイドで悪化する場合はカンジダを疑う

---

### Q4.「市販薬は何を使えばいい？」

**お母さん**「どんな薬を塗ればいいですか？」

**おかもん先生**「まずは**ワセリンや亜鉛華軟膏**で十分です [4]。」

| 薬 | 用途 |
|---|---|
| **ワセリン** | 皮膚の保護・バリア |
| **亜鉛華軟膏** | 乾燥・保護作用 |
| **弱いステロイド** | 赤みが強い場合（短期間） |
| **抗真菌薬** | カンジダの場合（医師の処方） |

| 使い方のポイント |
|---|
| 厚めに塗ってバリアを作る |
| 毎回全部拭き取らなくてよい |
| 便の後に塗り足す |

> **ポイント**
> - ワセリンが最も手軽で安全
> - 厚めに塗ってバリアを作る
> - ステロイドは短期間のみ

---

### Q5.「おむつの選び方は？」

**お母さん**「おむつかぶれしにくいおむつはありますか？」

**おかもん先生**「**通気性の良い吸収力の高いおむつ**を選びましょう [5]。」

| 選び方 | ポイント |
|---|---|
| **吸収力** | 表面がサラサラしているもの |
| **通気性** | 蒸れにくいもの |
| **サイズ** | きつすぎない、大きすぎない |
| **素材** | 肌に合うもの（合わなければ変える） |

| 布おむつと紙おむつ |
|---|
| 紙おむつ：吸収力が高く、おむつかぶれは少ない傾向 |
| 布おむつ：交換頻度を上げればOK。環境に配慮 |

> **ポイント**
> - 吸収力と通気性が大切
> - サイズが合っているか確認
> - 肌に合わなければ変える

---

## 今号のまとめ

- **おむつかぶれは湿気と刺激が原因**
- **こまめな交換とお湯で洗い流す**ことが基本
- **ワセリンで皮膚のバリアを作る**
- **1週間以上治らなければカンジダを疑う**
- **通気性の良いおむつ**を選ぶ
