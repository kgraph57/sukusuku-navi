# 愛育病院 小児科**おかもん先生** だより Vol.177

## 「頭がかゆい！」――アタマジラミの対処法

こんにちは。愛育病院小児科の**おかもん先生**です。

「園からアタマジラミが流行していると連絡がありました」。アタマジラミは不潔だから感染するのではなく、頭と頭の接触で感染します。集団生活をしている子どもなら誰でも感染する可能性があります。今回は、アタマジラミの見つけ方と対処法についてお伝えします。

---

### Q1.「アタマジラミとは？」

**お母さん**「シラミは不潔だからうつるのですか？」

**おかもん先生**「**シラミの感染は清潔とは関係ありません** [1]。毎日シャンプーしていても感染します。」

| 基本情報 | 詳細 |
|---|---|
| **大きさ** | 成虫は2-3mm（ゴマ粒大） |
| **卵（卵鞘）** | 0.5mm、白〜黄色、髪に固着 |
| **寿命** | 約30日。1日に3-4個産卵 |
| **感染経路** | 頭と頭の直接接触が主 |
| **好発年齢** | 3-12歳 |

| よくある誤解 | 正しい知識 |
|---|---|
| 「不潔だからうつる」 | 清潔さは関係ない |
| 「ジャンプして飛び移る」 | 飛べない。直接接触で移動 |
| 「病気をうつす」 | 日本のアタマジラミは病気を媒介しない |
| 「学校に行けない」 | 登校は可能（治療しながら） |

> **ポイント**
> - 不潔とは関係なく感染する
> - 頭と頭の直接接触が主な感染経路
> - 登校・登園は可能

---

### Q2.「アタマジラミの見つけ方は？」

**お母さん**「どうやって確認すればいいですか？」

**おかもん先生**「**卵を見つけるのが最も確実**です [2]。」

| チェック方法 | 手順 |
|---|---|
| **場所** | 明るい場所で確認 |
| **部位** | 耳の後ろ、えり足、側頭部を重点的に |
| **卵の特徴** | 白〜黄色、楕円形、髪に固着して動かない |
| **フケとの見分け** | フケは指で簡単に取れるが、卵は取れない |
| **成虫** | 動きが素早いので見つけにくい |

| 卵の見分けポイント |
|---|
| 髪の毛の根元から1cm以内に付着 → 生きている可能性が高い |
| 髪の毛の根元から1cm以上離れている → 孵化済みか死んでいる |
| 指で引っ張っても取れない → 卵（フケなら取れる） |

「**目の細かいクシ（すき櫛）**を使うと、成虫や卵を効率よく見つけられます。薬局で専用のすき櫛が購入可能です。」

> **ポイント**
> - 耳の後ろ、えり足を重点チェック
> - 卵はフケと違い指で取れない
> - すき櫛の使用が効果的

---

### Q3.「アタマジラミの治療法は？」

**お母さん**「どうすれば駆除できますか？」

**おかもん先生**「**駆除用シャンプーと物理的な除去**を組み合わせます [3]。」

| 治療法 | 方法 |
|---|---|
| **①駆除シャンプー** | スミスリンシャンプー（フェノトリン配合）。薬局で購入可能 |
| **②すき櫛** | 毎日、髪を濡らしてすき櫛で卵と成虫を除去 |
| **③洗濯** | シーツ、枕カバー、帽子を60℃以上で10分間洗濯 |

| スミスリンシャンプーの使い方 |
|---|
| 3日に1回、計4回（3日、6日、9日、12日目）使用 |
| 髪を濡らし、シャンプーを塗布して5分間放置 |
| 洗い流す |
| 卵には効かないため、3日おきに繰り返すことが重要 |

「スミスリンシャンプーは**卵には効かない**ため、孵化のタイミングに合わせて3日おきに4回使うことが重要です。**すき櫛による物理的な卵の除去**を毎日併用すると効果的です。」

> **ポイント**
> - スミスリンシャンプーは3日おき×4回
> - すき櫛で毎日卵を除去
> - シーツ等は60℃以上の熱水で洗濯

---

### Q4.「家族にうつらないようにするには？」

**お母さん**「兄弟にうつりそうで心配です」

**おかもん先生**「家庭内での対策をお伝えします [4]。」

| 対策 | 内容 |
|---|---|
| **家族全員のチェック** | 感染者が見つかったら家族全員の頭を確認 |
| **タオル・枕の個別使用** | 共有しない |
| **帽子・ヘアゴムの個別使用** | 共有しない |
| **寝具の洗濯** | 60℃以上の熱水で洗濯、または乾燥機（高温） |
| **掃除機** | ソファ、カーペットに掃除機をかける |
| **コートのフード** | 共有のロッカーでフードが接触しないよう注意 |

「**乾燥機の使用**も効果的です。55℃以上で10分間乾燥すればシラミは死滅します。洗濯できないものは、ビニール袋に入れて2週間放置すると、シラミは餓死します。」

> **ポイント**
> - 家族全員の頭をチェック
> - タオル、枕、帽子を共有しない
> - 乾燥機（高温）も効果的

---

### Q5.「登園・登校はできますか？」

**お母さん**「園に行っていいですか？」

**おかもん先生**「**治療を開始していれば登園・登校は可能**です [5]。」

| 学校保健 | 対応 |
|---|---|
| **出席停止** | アタマジラミは出席停止の対象ではない |
| **登園・登校** | 治療開始後は可能 |
| **園への報告** | 早めに伝えて集団感染を防ぐ |
| **プール** | タオルの共有を避ければ参加可能 |

「園や学校に伝えることは**恥ずかしいことではありません**。早めの報告が集団感染の拡大防止につながります。シラミは誰でも感染する可能性があり、不潔の証ではありません。」

> **ポイント**
> - 出席停止の対象ではない
> - 治療開始後は登園・登校OK
> - 早めに園・学校に報告を

## 今号のまとめ

- **アタマジラミは不潔とは関係なく感染**する
- **耳の後ろ、えり足の卵を確認**して早期発見
- **スミスリンシャンプー3日おき×4回＋すき櫛**で駆除
- **タオル等の共有を避け、60℃以上で洗濯**
- **治療開始後は登園・登校可能**
