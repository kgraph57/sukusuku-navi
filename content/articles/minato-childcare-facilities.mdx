# 愛育病院 小児科**おかもん先生** だより Vol.292

## 「子連れで行ける場所がほしい」――港区の子育て支援施設

こんにちは。愛育病院小児科の**おかもん先生**です。

「近所に子連れで気軽に行ける場所はありますか？」「子育ての悩みを相談したい」――港区には**子育て家庭を支援するさまざまな施設**があります。一人で抱え込まず、これらの施設を上手に活用してください。

---

### Q1.「港区の子育てひろばとは？」

**お母さん**「子育てひろばってどういう場所ですか？」

**おかもん先生**「子育てひろばは**0歳〜就学前の子どもと保護者が自由に遊べる場所**です [1]。予約不要で利用でき、スタッフに育児の相談もできます。」

| 施設名 | 場所 | 特徴 |
|---|---|---|
| **あっぴぃ芝浦** | 芝浦 | 広い室内遊び場 [1] |
| **あっぴぃ港南** | 港南 | 親子で遊べるスペース [1] |
| **あっぴぃ麻布** | 麻布 | アットホームな雰囲気 [1] |
| **あっぴぃ新橋** | 新橋 | 駅近で便利 [1] |
| **あっぴぃ白金台** | 白金台 | 自然豊かなエリア [1] |

| 子育てひろばの特徴 |
|---|
| **利用料無料** |
| **予約不要**（一部イベントは予約制） |
| 保育士やスタッフが常駐 |
| 育児相談が可能 |
| 季節のイベントや講座を開催 |
| 同年代の子を持つ保護者同士の交流の場 |

> **ポイント**
> - **無料・予約不要**で気軽に利用可能 [1]
> - スタッフに**育児相談**ができる
> - **ママ友・パパ友づくり**の場としても

---

### Q2.「子ども家庭支援センターとは？」

**お母さん**「子ども家庭支援センターではどんな支援が受けられますか？」

**おかもん先生**「子ども家庭支援センターは**18歳未満の子どもに関するあらゆる相談**に対応する施設です [2]。」

| 項目 | 内容 |
|---|---|
| **対象** | 18歳未満の子どもとその家族 [2] |
| **費用** | 無料 |
| **相談内容** | 育児の悩み、発達、しつけ、家族関係、虐待等 [2] |
| **相談方法** | 来所、電話、オンライン |

| 主なサービス |
|---|
| 育児・子育て全般の相談 |
| ショートステイ（子どもの一時預かり） |
| トワイライトステイ（夜間の預かり） |
| ファミリー・サポート事業の紹介 |
| 子育てに関する情報提供 |

> **ポイント**
> - **18歳未満のあらゆる相談**に対応 [2]
> - 一時預かり（ショートステイ）も利用可能
> - 電話やオンラインでも相談可能

---

### Q3.「一時預かりサービスはありますか？」

**お母さん**「リフレッシュや急用の時に預けられる場所はありますか？」

**おかもん先生**「港区にはいくつかの一時預かりサービスがあります [1]。」

| サービス | 対象 | 特徴 |
|---|---|---|
| **一時保育** | 生後4か月〜就学前 | 保育園での一時預かり [1] |
| **ショートステイ** | 0〜18歳未満 | 宿泊を伴う預かり [2] |
| **トワイライトステイ** | 0〜18歳未満 | 夜間の預かり [2] |
| **ファミリー・サポート** | 生後2か月〜小学6年生 | 地域の援助者が預かり [1] |
| **ベビーシッター利用支援** | 0歳〜就学前 | ベビーシッター費用の助成 [1] |

| 一時保育の利用方法 |
|---|
| 事前に登録が必要 |
| 利用日の前月に予約 |
| 理由は問わない（リフレッシュもOK） |
| 1日の利用料は施設により異なる |

> **ポイント**
> - **リフレッシュ目的**でも一時保育を利用できる [1]
> - **ファミリー・サポート**は地域の支え合いの仕組み
> - ベビーシッター費用の助成制度もある [1]

---

### Q4.「子育てサークルや講座はありますか？」

**お母さん**「他のお母さんたちとつながりたいのですが」

**おかもん先生**「港区ではさまざまな子育て講座やサークルがあります [1]。」

| 講座・イベント | 内容 |
|---|---|
| **プレママ・プレパパ講座** | 出産前の準備、育児の基礎 [1] |
| **離乳食講習会** | 離乳食の作り方、進め方 [1] |
| **育児学級** | 月齢に応じた育児のポイント |
| **多胎児交流会** | 双子・三つ子の保護者同士の交流 |
| **パパの子育て講座** | 父親向けの育児プログラム |

| 情報を得る方法 |
|---|
| 港区公式ウェブサイト |
| 港区子育て応援アプリ |
| 各子育てひろばの掲示板 |
| みなと保健所からのお知らせ |
| 区の広報紙「広報みなと」 |

> **ポイント**
> - **離乳食講習会**や**育児学級**は非常に人気 [1]
> - 情報は**港区公式サイトと子育て応援アプリ**でチェック
> - パパ向けの講座もある

---

### Q5.「外国人の保護者向けのサポートはありますか？」

**お母さん**「英語での対応はありますか？」

**おかもん先生**「港区は外国人居住者が多い地域であり、多言語対応の支援があります [3]。」

| サービス | 内容 |
|---|---|
| **国際交流ラウンジ** | 外国人向けの情報提供、生活相談 [3] |
| **多言語の子育て情報** | 英語・中国語等での情報発信 [3] |
| **通訳サービス** | 健診等での通訳支援 [3] |
| **外国人向け子育てサロン** | 外国人保護者同士の交流の場 [3] |

| 対応言語 |
|---|
| 英語、中国語、韓国語等 |
| やさしい日本語での情報提供も |

> **ポイント**
> - 港区は**多言語対応の子育て支援**が充実 [3]
> - **国際交流ラウンジ**が外国人の方の窓口
> - 健診時の**通訳支援**も利用可能

## 今号のまとめ

- 港区の**子育てひろば**は無料・予約不要で利用可能
- **子ども家庭支援センター**で18歳未満のあらゆる相談に対応
- **一時預かり**はリフレッシュ目的でもOK
- **離乳食講習会**や**育児学級**など講座も充実
- 外国人向けの**多言語対応**もあり
