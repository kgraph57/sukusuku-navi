# キャラクター設定メモ

## スタイル共通ルール
- 水彩・絵本タッチ（newsletter-illustrationスキル準拠）
- ふわっとした輪郭、柔らかいパステル系カラー
- 丸みのある可愛いデフォルメ体型
- 白背景（透過PNG用）
- ズートピアのような「生き生きとした世界観・シーン感」を表現

## キャラクター一覧

### コンコン（キツネ）
- オレンジ色の毛、白いお腹、クリーム色のVネックセーター、ジーンズ
- ふわふわの大きなしっぽ
- 優しい先生キャラ、聴診器を持つことも

### うさぎーさん（ウサギ）
- 白いふわふわのウサギ、紫のオーバーオール
- ピンクの耳の内側
- 表情豊か：CHEERFUL / THINKING / LAUGHING / CALM / ENCOURAGING

### ぱんくん（パンダ）
- 白黒のパンダ、ミントグリーンのTシャツ、グレーのスウェットパンツ
- ぽっちゃり体型、穏やか
- 表情：PEACEFUL / SLEEPY / GENTLE / HAPPY / EATING

### わんぱくキッズ - BOYS
- ハルト：黒髪、丸メガネ、緑のオーバーオール、ストライプTシャツ
- レオ：ウェービーブロンドヘア、コーラルレッドのパーカー、ショートパンツ

### わんぱくキッズ - GIRLS
- アマラ：ダークスキン、カラフルなヘアポンポン、黄色の水玉ドレス、オレンジのレギンス
- ミア：カーリーブラウンヘア、セージグリーンのトップス（花柄刺繍）、紫のパンツ
