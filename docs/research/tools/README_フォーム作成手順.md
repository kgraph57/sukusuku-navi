# Googleフォーム作成手順

## 所要時間: 5分

---

## ステップ1: スクリプト実行（2分）

1. ブラウザで https://script.google.com を開く
2. 「新しいプロジェクト」をクリック
3. エディタに表示されるコードをすべて削除
4. `create_question_form.gs` の中身をすべてコピー＆ペースト
5. 上部の「▶ 実行」ボタンをクリック
6. 「承認が必要です」→「アカウントを選択」→「許可」
7. 実行が完了すると、下部の「実行ログ」に以下が表示される:
   - **フォームURL**（読者に共有するURL）
   - **スプレッドシートURL**（回答が記録されるシート）

> ⚠️ 「このアプリは確認されていません」と出る場合:
> 「詳細」→「〇〇（安全でないページ）に移動」で進めてOK（自分のスクリプトなので安全）

---

## ステップ2: URL短縮 & QRコード（2分）

### URL短縮
1. https://bit.ly にアクセス（アカウント不要で使える）
2. フォームURLを貼り付けて短縮
3. 例: `bit.ly/okamon-question`

### QRコード生成
1. https://www.qrcode-monkey.com にアクセス
2. 「Your URL」にフォームURLを入力
3. 「Create QR Code」→ ダウンロード（PNG）
4. 同様に、theLetter登録URLのQRコードも生成

---

## ステップ3: ポスター完成（1分）

1. `待合室ポスター.html` をブラウザで開く
2. Cmd+P（印刷）→ A4で印刷
3. QRコード部分に、印刷したQRコードを貼付
   - 左: theLetter登録QR
   - 右: 質問フォームQR

> 💡 QRコード画像をHTMLに直接埋め込みたい場合:
> `待合室ポスター.html` の `<div class="qr-placeholder">` を
> `<img src="qr.png" width="120">` に差し替えてください

---

## ステップ4: 記事にCTAを挿入

1. `CTA_テンプレート.md` を開く
2. 記事フォーマットに合ったパターン（A/B/C）をコピー
3. `★フォームURL★` を実際の短縮URLに置換
4. 記事の署名の直前に貼り付け

---

## 運用の流れ

```
週1回: スプレッドシートを確認
  ↓
質問を「管理ステータス」シートに転記
  ↓
ステータスを「検討中」に変更
  ↓
月1回: 3〜5問を選んで【読者のギモン回答】として配信
  ↓
配信後、ステータスを「掲載済（volXXX）」に更新
```
