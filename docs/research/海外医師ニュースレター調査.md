# 海外医師ニュースレター コンテンツ形式調査

> 調査日: 2026-02-09
> 目的: おかもんだよりのコンテンツ多様化の参考資料

---

## 1. Dr. Lucy McBride「Are You Okay?」

- **URL**: https://lucymcbride.substack.com/
- **読者数**: 31,000人以上（週次配信）
- **専門**: プライマリケア（ハーバード大出身）
- **書籍化**: Simon & Schuster から2026年出版予定

### コンテンツ形式

| 形式 | 頻度 | 内容 |
|---|---|---|
| **Weekly Essay** | 毎週 | 1テーマをエッセイ形式で深掘り。エビデンスベースだが個人の視点・語りを重視 |
| **Friday Q&A** | 毎週金曜 | 読者から寄せられた質問に回答。2025年から全読者に無料開放 |
| **Live Q&A "The Doctor Is In"** | 毎週 | リアルタイムで読者の質問に答えるライブ配信。有料会員のみ質問可能 |
| **Podcast「Beyond the Prescription」** | 定期 | ゲストインタビュー形式。患者のように話を引き出すスタイル |
| **Community Chat** | 常時 | 有料会員限定のコミュニティ空間（Substack Chat機能） |

### おかもんへの示唆
- **Q&Aは週1の定番コラムとして独立させる**手法が有効
- エッセイ形式（おかもんの語り）とQ&A（読者対話）を分けている
- ライブ配信は双方向性の強化に有効（theLetter対応は要確認）

## 2. Emily Oster「ParentData」

- **URL**: https://parentdata.org/
- **規模**: 著書累計100万部以上、ニュースレター読者数非公開だが大規模
- **専門**: 経済学者（ハーバード大→ブラウン大）。医師ではないがデータ分析の観点で育児を語る
- **特徴**: 「データで育児の意思決定を支える」がコンセプト

### コンテンツ形式

| 形式 | 内容 |
|---|---|
| **The Latest** | Emilyが最新の研究・ニュースヘッドラインを解説 |
| **Trying to Conceive (TTC)** | 不妊治療・家族計画のトピック |
| **PregnantData** | 出産予定日に基づいてパーソナライズされた妊娠ガイド（0-40週+） |
| **BabyData** | 0-18ヶ月の育児インサイト |
| **ToddlerData** | 18-22ヶ月の15週構造化ガイド |
| **Digest** | 週間まとめ（TTC、妊娠、育児の全トピック横断） |
| **Ask Emily** | 読者からの質問を募集して回答するポータル |
| **Petey AI** | 過去1,300+記事を検索できるAIアシスタント（ライブラリアン） |

### サイト構造（年齢×トピック）
- **ライフステージ別**: TTC → 妊娠 → 赤ちゃん → トドラー → キッズ → 大人
- **トピック別**: 母乳、栄養、ウェルネス、睡眠、スクリーン、トイトレ等

### おかもんへの示唆
- **年齢別にコンテンツを整理する**発想は、健診シリーズ（1ヶ月、3-4ヶ月、6-7ヶ月…）と相性がよい
- Digestは「今週のまとめ」として過去記事への再導線に使える
- Ask Emily的な「読者質問募集→回答」は双方向性の核

## 3. Dr. Matt Hale「Kids Health Decoded」

- **URL**: https://drmattpaed.substack.com/
- **専門**: 小児科医（おかもんと最も近い立場）
- **特徴**: 月次研究サマリー＋エビデンス強度ラベルが独自

### コンテンツ形式

| 形式 | 頻度 | 内容 |
|---|---|---|
| **Child Health Brief** | 月1回 | 最新研究を5本ピックアップし、要約＋親向けアクション＋医療者向けメモを付記 |
| **Myth-Buster of the Month** | 月1回 | 通説を1つ取り上げてエビデンスで検証 |
| **Deep Dive** | 不定期 | ADHDなど1テーマを徹底解説 |
| **Practical Guide** | 不定期 | 病気別の実用ガイド |
| **Essays** | 不定期 | 政治・テクノロジー・社会トレンドが子どもの健康に与える影響 |
| **Clinical Resources** | 不定期 | 医療者向けの神経発達ケアガイド |

### Child Health Briefの構造（特に参考になる）
1. 導入（今月のテーマ概要）
2. トピック×5本（各トピックに以下を付記）
   - **エビデンス強度ラベル**: 「Strong」「Emerging」「Needs More Data」
   - **For Parents**: 親向けのアクション
   - **For Clinicians**: 医療者向けメモ
3. **Myth-Buster of the Month**: 通説検証コーナー
4. **Ask-Your-Doctor Checklist**: 受診時にそのまま使えるチェックリスト
5. 参考文献一覧

### おかもんへの示唆
- **エビデンス強度ラベル**は読者の理解度を上げる工夫として秀逸
- **Myth-Buster**は企画書の「通説検証シリーズ」と完全に合致
- **Ask-Your-Doctor Checklist**は「受診時に見せる1枚シート」として差別化要素になる
- Child Health Brief的な月次まとめは、最新の小児医療ニュースを届ける箱として有用

## 4. Jeremy Faust「Inside Medicine」

- **URL**: https://insidemedicine.substack.com/
- **専門**: 救急医（ER医師）、ハーバード大ブリガム病院
- **特徴**: 臨床現場のリアルタイム観察×データ分析。「親からCDC長官まで」が読者層

### コンテンツ形式

| 形式 | 内容 |
|---|---|
| **Deep Dive Analysis** | 最新論文をデータごと読み解く。他メディアより先に結論を出すことも |
| **Field Notes** | 自分のERでの症例を匿名化して共有。「こんな患者さんが来ました」形式 |
| **Quick Takes** | 短い速報的コメント。旬のトピックへの所見 |
| **Follow-up** | 過去記事のアップデート。新データが出たら再訪 |
| **Interview** | 医療界のキーパーソンとの対談（有料） |

### おかもんへの示唆
- **Field Notes（外来メモ）** は最もおかもんらしい形式になり得る。「今週の外来でこんな相談がありました」→匿名化して学びを共有
- **Follow-up**の発想は、季節の変わり目に過去記事を更新・再配信する際に使える
- **Quick Takes**は感染症の流行速報に向いている

## おかもんだよりへの導入候補まとめ

### 現状: Q&A対話形式のみ（vol001-024）

### 追加候補フォーマット（優先度順）

| # | フォーマット名 | 参考元 | 頻度案 | 内容 | 導入しやすさ |
|---|---|---|---|---|---|
| 1 | **今月の通説検証** | Dr. Matt | 月1回 | 「タミフルで異常行動」「B型は軽い」等の通説を1つ取り上げてエビデンスで検証 | 高（既存の「誤解シリーズ」を独立コーナー化） |
| 2 | **おかもんの外来メモ** | Faust | 月1-2回 | 匿名化した外来エピソードから学びを共有。「こんな相談がありました」形式 | 高（実際の外来経験がそのまま素材） |
| 3 | **読者のギモン回答** | McBride / Oster | 月1回 | 読者から募集した質問にまとめて回答 | 中（読者基盤ができてから） |
| 4 | **最新ニュース解説** | Oster / Faust | 随時 | 報道された小児医療ニュースを速報的に解説 | 中（タイムリーさが求められる） |
| 5 | **受診チェックリスト** | Dr. Matt | Q&Aに付録 | 記事の要点を「受診時に見せる1枚シート」として付録 | 高（既存記事に追加するだけ） |
| 6 | **数字で見るシリーズ** | Oster | 隔月 | 「日本の子ども10人に1人が食物アレルギー」等、データから切り込む | 中（データ収集に手間） |

### 推奨: まず導入する2つ

1. **今月の通説検証**（Myth-Buster）: 企画書にも「通説検証シリーズ」として既に構想あり。Q&Aとは明確にトーンが違い、バリエーションが出る
2. **おかもんの外来メモ**（Field Notes）: おかもんの人柄と臨床経験が最も出るフォーマット。AIには書けないオリジナリティ

## 配信スケジュール案（月4回配信の場合）

| 週 | フォーマット |
|---|---|
| 第1週 | **Q&A対話**（メインコンテンツ） |
| 第2週 | **おかもんの外来メモ** or **最新ニュース解説** |
| 第3週 | **Q&A対話**（メインコンテンツ） |
| 第4週 | **今月の通説検証** or **読者のギモン回答** |
