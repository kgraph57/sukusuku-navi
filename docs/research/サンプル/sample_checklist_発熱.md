## 【受診チェックリスト】お子さんが発熱した時に

> このチェックリストは、受診時にスマホで見せたり、印刷して持っていくことを想定しています。

---

### いつ受診する？

以下に当てはまるものがあれば、チェックしてください。

#### すぐに受診（救急）

- [ ] **生後3ヶ月未満**で38℃以上の熱がある
- [ ] ぐったりして**呼びかけに反応が弱い**
- [ ] **けいれん**が起きた（または5分以上続いている）
- [ ] **呼吸が苦しそう**（肩で息をしている、胸がペコペコへこむ）
- [ ] **顔色が悪い**（土色、青白い、唇が紫色）
- [ ] **水分がまったくとれない**
- [ ] 赤い発疹が出て**押しても消えない**（紫斑）

> ⚠️ **上の項目に1つでも当てはまる場合は、すぐに受診してください。夜間・休日でも迷わず救急へ。**

#### 翌日の日中に受診

- [ ] 熱が**4日以上**続いている
- [ ] **耳を痛がる**、耳だれが出る
- [ ] 咳が**どんどんひどくなっている**（眠れないほど）
- [ ] **おしっこの回数が明らかに減っている**
- [ ] **発疹**が出てきた
- [ ] **機嫌が悪い状態が続く**（あやしても泣き止まない）

#### 様子を見てOK

- [ ] 熱はあるが、**水分がとれていて、機嫌もまずまず**
- [ ] 鼻水・軽い咳があるが、**呼吸は楽そう**
- [ ] 熱は38℃台で、**2〜3日以内に下がってきた**

### 家庭でできるケアの要点

| やること | 具体的に |
|---|---|
| **水分補給** | 経口補水液・お茶・母乳/ミルクを少量ずつ頻回に |
| **室温調整** | 22〜24℃程度。寒がっていなければ薄着でOK |
| **解熱剤の使い方** | 38.5℃以上でつらそうな時に。元気なら不要 |
| **食事** | 食べられるものを食べられる量で。無理強いしない |
| **観察ポイント** | おしっこの回数・機嫌・呼吸の様子 |
