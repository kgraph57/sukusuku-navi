# イラスト図解マスタープラン：すくすくナビ全60記事

## 概要

「教えて！ドクター」（佐久医師会）のように、**イラスト1枚で「見てわかる」コンテンツ**を既存記事に追加する。
テキスト＋Q&A対話形式の記事に、**視覚的に伝わる図解**を組み込むことで、読了率と理解度を向上させる。

## 共通スタイルガイド

- **画風**: 水彩タッチ、パステルカラー、輪郭やわらか、背景白or透過
- **キャラ**: コンコン先生（きつね/医師）、うさぎさん（母親）、くまくん（見守り）、パンくん（好奇心）
- **カラーパレット**: teal系（信頼）、coral系（温かみ）、sage系（安心）
- **フォーマット**: PNG透過、2x Retina
- **格納先**: `/public/characters/illustrations/articles/` （記事用イラスト専用ディレクトリ）

## イラスト種類

| 種類 | 説明 | サイズ目安 |
|------|------|-----------|
| **フローチャート** | 判断分岐を図解 | 600×400px |
| **比較図** | ○○ vs △△の見た目・特徴比較 | 600×350px |
| **ステップ図** | 1→2→3の手順 | 600×300px |
| **部位マップ** | 体のどこに何が起きるか | 400×500px |
| **タイムライン** | 経過・時系列 | 700×250px |
| **チェックリスト図** | 視覚的なチェック項目 | 500×400px |
| **シーンイラスト** | 状況を伝えるワンシーン | 600×400px |
